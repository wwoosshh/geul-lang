# 글 프로그래밍 언어

한글로 프로그래밍하는 언어입니다.

## 설치

### 방법 1: 원커맨드 설치 (PowerShell)
```powershell
irm https://raw.githubusercontent.com/wwoosshh/geul-lang/main/install.ps1 | iex
```

### 방법 2: 수동 설치
1. [최신 릴리즈](https://github.com/wwoosshh/geul-lang/releases/latest)에서 `geul-language-0.1.0.vsix` 다운로드
2. VS Code → `Ctrl+Shift+P` → "VSIX에서 설치" → 파일 선택

## 사용법

`.글` 파일을 만들고 코드를 작성한 뒤 **F5**를 누르면 실행됩니다.

```
[시작하기]는 {
    "안녕하세요!\n"을 쓰기다.
    "1 + 2 = %d\n"을 (1 + 2)를 쓰기다.
}
```

## 기능

| 단축키 | 기능 |
|--------|------|
| **F5** | 컴파일 + 실행 |
| **Ctrl+Shift+B** | 빌드 (.exe 생성) |

## 문법 요약

```
(* 변수 *)
정수 나이는 25.
문자열 이름은 "홍길동".

(* 함수 *)
[정수 반지름을 원넓이]는 -> 정수 {
    반환 반지름 * 반지름 * 3.
}

(* 조건문 *)
나이 >= 20이면 {
    "성인\n"을 쓰기다.
}

(* 반복문 *)
정수 번호 = 0.
(번호 < 10)동안 {
    번호++다.
}
```

- `정수`, `문자열`, `참거짓` — 타입
- `쓰기` — 화면 출력
- `반환` — 값 반환
- `이면` / `아니면` — 조건문
- `동안` — 반복문
- `묶음` — 구조체
- `포함` — 다른 파일 포함

## 요구사항

- VS Code
- Windows 10/11 (다른 OS 지원 예정)
