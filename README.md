# 글 프로그래밍 언어

**한글로 프로그래밍하는 언어입니다.**

```글
[시작하기]는 {
    "안녕하세요! 글 언어입니다.\n"을 쓰기다.
}
```

---

## 설치

### 방법 1: 원커맨드 설치 (PowerShell)
```powershell
irm https://raw.githubusercontent.com/wwoosshh/geul-lang/main/install.ps1 | iex
```

### 방법 2: 수동 설치
1. [최신 릴리즈](https://github.com/wwoosshh/geul-lang/releases/latest)에서 `geul-language-0.2.0.vsix` 다운로드
2. VS Code → `Ctrl+Shift+P` → "VSIX에서 설치" → 파일 선택

---

## 사용법

1. VS Code에서 아무 폴더 열기
2. 새 파일 만들기 → `.글`로 저장
3. 코드 작성 → **F5** → 실행

| 단축키 | 기능 |
|--------|------|
| **F5** | 컴파일 + 실행 |
| **Ctrl+Shift+B** | 빌드 (.exe 생성) |

## 예제

### 계산기
```글
[정수 왼쪽을 정수 오른쪽을 더하기]는 -> 정수 {
    반환 왼쪽 + 오른쪽.
}

[시작하기]는 {
    "10 + 20 = %d\n"을 더하기(10, 20)을 쓰기다.
}
```

### 반복문
```글
[시작하기]는 {
    정수 합계 = 0.
    반복 (정수 번호 = 1: 번호 <= 100: 번호++) {
        합계 += 번호.
    }
    "1~100 합: %d\n"을 합계를 쓰기다.
}
```

### 재귀 함수
```글
[정수 숫자를 팩토리얼]은 -> 정수 {
    숫자 <= 1이면 { 반환 1. }
    반환 숫자 * 팩토리얼(숫자 - 1).
}

[시작하기]는 {
    "10! = %d\n"을 팩토리얼(10)을 쓰기다.
}
```

## 📖 문법 안내서

**[글 언어 문법서 전체 보기](docs/grammar.md)**

- 데이터 타입 (정수, 실수, 문자열, ...)
- 변수, 상수, 연산자
- 함수 정의와 호출 (SOV 어순 + 조사)
- 조건문, 반복문, 갈래문
- 배열, 구조체, 열거형
- 포인터와 메모리 관리
- 표준 라이브러리 함수 목록
- 전처리기 (포함, 정의, 조건부 컴파일)
- 고급 기능 (함수 포인터, 가변 인자, 합침)

## 핵심 특징

- **100% 한글** — 변수명을 제외한 모든 키워드가 한국어
- **SOV 어순** — 한국어 문법처럼 `대상을 함수하다`
- **조사 바인딩** — `을/를`, `에`, `로` 등 조사가 인자 역할 결정
- **자동 표준 라이브러리** — `포함` 없이 바로 `쓰기`, `할당` 등 사용 가능
- **VS Code 통합** — 문법 강조, F5 실행, 코드 자동완성
