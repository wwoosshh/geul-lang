# 글 프로그래밍 언어

**한글로 프로그래밍하는 독자적 프로그래밍 언어입니다.**

```글
포함 "std.gl"

[시작하기]는 -> 정수 {
    문자열 이름 = "세계".
    정수 년도 = 2026.
    "안녕, {이름}! {년도}년입니다.\n"을 쓰다.
    반환 0.
}
```

---

## 특징

- **100% 한글 키워드** — 영어 함수명이 사용자 코드에 노출되지 않음
- **SOV 어순** — 한국어 문법처럼 `대상을 함수하다`
- **조사 바인딩** — `을/를`, `에`, `로` 등 조사가 인자 역할 결정
- **문자열 보간** — `"이름: {이름}, 나이: {나이}세"` 변수 직접 삽입
- **네이티브 코드 직접 생성** — C 컴파일러 없이 `.글 → .exe` 직접 컴파일
- **자체호스팅** — 글 컴파일러가 자기 자신을 컴파일
- **VS Code 통합** — 문법 강조, F5 실행, LSP 자동완성

## 성능

> C는 MSVC `/O2` 최적화 적용. 측정: Intel Ultra 5 226V, 5회 중앙값.

| 벤치마크 | 글 네이티브 | C (/O2) | 배율 | Python | 배율 |
|---------|----------:|-------:|-----:|-------:|-----:|
| 재귀 fib(40) | 464 ms | 321 ms | 1.45x | 7,364 ms | 23.0x |
| 소수 체 100만 | 10 ms | 8 ms | 1.30x | 167 ms | 22.0x |
| 버블정렬 3만 | 1,564 ms | 564 ms | 2.77x | 38,377 ms | 68.0x |

**C(/O2) 대비 1.3~2.8x** — Python 대비 22~68x 빠름

## 설치

### 방법 1: 원커맨드 설치 (PowerShell)
```powershell
irm https://raw.githubusercontent.com/wwoosshh/geul-lang/main/install.ps1 | iex
```
설치 후 **새 터미널**을 열어야 PATH가 적용됩니다.

### 방법 2: VS Code 확장 수동 설치
1. [릴리즈](https://github.com/wwoosshh/geul-lang/releases)에서 `.vsix` 다운로드
2. VS Code → `Ctrl+Shift+P` → "VSIX에서 설치" → 파일 선택

### 방법 3: 소스에서 빌드
```bash
# 부트스트랩 컴파일러 빌드 (MSVC 필요)
powershell -File 부트스트랩/build_temp.ps1

# 네이티브 컴파일러 빌드
부트스트랩/build/글cc.exe 자체호스팅/네이티브컴파일러.글
```

## 문법 가이드

### 기본 구조

```글
포함 "std.gl"

[시작하기]는 -> 정수 {
    "안녕하세요!\n"을 쓰다.
    반환 0.
}
```

- `포함` — 표준 라이브러리 포함 (#include)
- `[시작하기]` — 프로그램 진입점 (main)
- `.` — 문장 종결 (세미콜론 대신)
- `(* ... *)` — 주석

### 변수와 타입

```글
정수 나이 = 25.                  (* int *)
실수 원주율 = 3.14159.           (* double *)
문자열 이름 = "홍길동".          (* char* *)
문자 첫글자 = 'A'.               (* char *)
참거짓 활성 = 참.                (* bool *)
긴정수 큰값 = 1000000000.       (* long long *)
상수 정수 최대값 = 100.          (* const int *)
```

### 배열

```글
정수[10] 점수.                   (* int 점수[10]; *)
점수[0] = 95.
점수[1] = 87.
정수 첫점수 = 점수[0].

문자[256] 버퍼.                  (* char 버퍼[256]; *)
```

### 함수 정의와 호출

```글
(* 함수 정의 *)
[정수 왼쪽을 정수 오른쪽에 더하기]는 -> 정수 {
    반환 왼쪽 + 오른쪽.
}

(* SOV 호출 — 한국어 어순 *)
정수 결과는 5를 3에 더하다.

(* 동사형 호출 — 더하다 → 더하기 자동 매칭 *)
정수 결과2는 10을 20에 더하다.

(* 직접 호출 — C 스타일 *)
정수 결과3 = 더하기(5, 3).
```

> **동사형 호출**: `쓰다`는 `쓰기` 함수를, `더하다`는 `더하기` 함수를 자동으로 찾습니다.

### 문자열 보간

```글
문자열 이름 = "철수".
정수 나이 = 25.
정수 점수 = 95.

"이름: {이름}, 나이: {나이}세\n"을 쓰다.
"{이름}의 점수는 {점수}점입니다\n"을 쓰다.
```

출력:
```
이름: 철수, 나이: 25세
철수의 점수는 95점입니다
```

변수 타입에 따라 자동으로 형식이 결정됩니다:
| 타입 | 형식 |
|------|------|
| `정수` | `%d` |
| `문자열` | `%s` |
| `실수` | `%f` |
| `문자` | `%c` |
| `긴정수` | `%lld` |

기존 printf 스타일 `"값: %d\n"을 값을 쓰다.`도 동일하게 사용 가능합니다.

### 조건문

```글
(* 기본 조건문 *)
값 > 10이면 {
    "10보다 큽니다\n"을 쓰다.
}
아니면 {
    "10 이하입니다\n"을 쓰다.
}

(* 비교 연산자: ==, !=, <, >, <=, >= *)
이름이 "철수"와 같으면 {
    "철수입니다\n"을 쓰다.
}
```

### 반복문

```글
(* while 루프 *)
정수 번호 = 0.
(번호 < 10)동안 {
    "%d\n"을 번호를 쓰다.
    번호를 증가하다.
}

(* C-style for 루프 — 구분자는 콜론 *)
반복 (정수 순번 = 0: 순번 < 5: 순번++) {
    "%d\n"을 순번을 쓰다.
}

(* do-while 루프 *)
반복하기 {
    번호를 증가하다.
} 번호 < 100 동안.

(* 루프 제어 *)
(참)동안 {
    번호 >= 50이면 { 탈출. }     (* break *)
    번호 % 2 == 0이면 { 계속. }  (* continue *)
}
```

### 증가/감소

```글
번호를 증가하다.                 (* 번호++ *)
번호를 감소하다.                 (* 번호-- *)
번호++.                          (* C-style 후위 증가 *)
++번호.                          (* C-style 전위 증가 *)
```

### 복합 대입

```글
합계 += 10.                      (* 합계 = 합계 + 10 *)
값 -= 5.
값 *= 2.
값 /= 3.
값 %= 7.
```

### 삼항 연산자

```글
정수 결과 = 값 > 0 ? 값 : 0 - 값.   (* 절대값 *)
```

### 갈래문 (switch/case)

```글
갈래 (값) {
    경우 1: {
        "하나\n"을 쓰다.
    }
    경우 2: {
        "둘\n"을 쓰다.
    }
    기본: {
        "기타\n"을 쓰다.
    }
}
```

### 구조체 (묶음)

```글
묶음 점은 정수 가로, 정수 세로.

[시작하기]는 -> 정수 {
    점 위치.
    위치의 가로 = 10.
    위치의 세로 = 20.
    "위치: (%d, %d)\n"을 위치의 가로를 위치의 세로를 쓰다.
    반환 0.
}
```

### 열거형 (나열)

```글
나열 색깔은 빨강, 초록, 파랑.

[시작하기]는 -> 정수 {
    정수 내색 = 초록.
    "색깔: %d\n"을 내색을 쓰다.    (* 1 *)
    반환 0.
}
```

### 공용체 (합침)

```글
합침 값저장은 정수 정수값, 실수 실수값, 문자열 문자열값.
```

### 타입 별칭

```글
별칭 크기는 긴정수.               (* typedef long long 크기; *)
크기 파일크기 = 1048576.
```

### 포인터와 메모리

```글
정수 값 = 42.
정수* 주소 = &값.                 (* 주소 참조 *)
정수 복사 = *주소.                (* 역참조 *)

(* 동적 메모리 *)
문자열 버퍼 = 1024를 할당하다.    (* malloc *)
버퍼를 해제하다.                  (* free *)

(* 포인터 멤버 접근 *)
점* 참조 = &위치.
참조→가로 = 30.                  (* -> 멤버 접근 *)
```

### 형변환

```글
정수 정수값 = 65.
문자 문자값 = 정수값으로 문자.     (* (char)정수값 *)
실수 실수값 = 정수값으로 실수.     (* (double)정수값 *)
```

### 함수 포인터

```글
[정수, 정수 -> 정수] 연산.        (* int (*연산)(int, int) *)
연산 = 더하기.
정수 결과 = 연산(3, 5).
```

### 가변인자 함수

```글
[문자열 형식을 ... 내출력]은 {
    가변목록 목록.
    형식을 목록을 가변시작다.
    형식을 목록을 vprintf다.
    목록을 가변끝다.
}
```

### 정적 변수/함수

```글
정적 정수 호출횟수 = 0.           (* static int *)

정적 [카운트증가]는 {             (* static 함수 *)
    호출횟수를 증가하다.
}
```

### 전처리기

```글
포함 "std.gl"                     (* #include *)
정의 최대크기 1024                 (* #define *)
외부 정수 전역값.                  (* extern *)

만약정의 디버그                    (* #ifdef *)
    "디버그 모드\n"을 쓰다.
아니면                             (* #else *)
    "릴리즈 모드\n"을 쓰다.
끝                                 (* #endif *)
```

### 크기 연산자

```글
정수 바이트 = 크기(정수).          (* sizeof(int) *)
```

## 문법 요약표

| 문법 | 글 | C 대응 |
|------|-----|--------|
| 변수 선언 | `정수 값은 42.` | `int 값 = 42;` |
| 상수 | `상수 정수 값 = 42.` | `const int 값 = 42;` |
| 함수 정의 | `[정수 수를 제곱]은 -> 정수 { }` | `int 제곱(int 수) { }` |
| SOV 호출 | `42를 제곱하다.` | `제곱(42);` |
| 직접 호출 | `제곱(42).` | `제곱(42);` |
| 반환 | `결과를 반환하다.` | `return 결과;` |
| 문자열 보간 | `"{이름}은 {나이}세"` | *(printf 자동 생성)* |
| 증가/감소 | `번호를 증가하다.` | `번호++;` |
| 복합대입 | `값 += 10.` | `값 += 10;` |
| 삼항 | `조건 ? 참 : 거짓` | `조건 ? 참 : 거짓` |
| 조건문 | `값 > 10이면 { }` | `if (값 > 10) { }` |
| 아니면 | `아니면 { }` | `else { }` |
| while | `(조건)동안 { }` | `while (조건) { }` |
| for | `반복 (초기화: 조건: 증감) { }` | `for (초기화; 조건; 증감) { }` |
| do-while | `반복하기 { } 조건 동안.` | `do { } while (조건);` |
| switch | `갈래 (값) { 경우 1: { } }` | `switch (값) { case 1: { } }` |
| 탈출 | `탈출.` | `break;` |
| 계속 | `계속.` | `continue;` |
| 구조체 | `묶음 이름은 필드들.` | `struct 이름 { 필드들 };` |
| 열거형 | `나열 이름은 값들.` | `enum 이름 { 값들 };` |
| 공용체 | `합침 이름은 필드들.` | `union 이름 { 필드들 };` |
| 별칭 | `별칭 새이름은 기존타입.` | `typedef 기존타입 새이름;` |
| 멤버접근 | `객체의 필드` | `객체.필드` |
| 포인터접근 | `참조→필드` | `참조->필드` |
| 형변환 | `값으로 타입` | `(타입)값` |
| sizeof | `크기(타입)` | `sizeof(타입)` |
| 포함 | `포함 "파일"` | `#include "파일"` |
| 정의 | `정의 이름 값` | `#define 이름 값` |
| 조건컴파일 | `만약정의 이름 ... 끝` | `#ifdef 이름 ... #endif` |
| 주석 | `(* 내용 *)` | `/* 내용 */` |

## 예제

### 구구단

```글
포함 "std.gl"

[시작하기]는 -> 정수 {
    정수 단 = 2.
    (단 <= 9)동안 {
        정수 곱 = 1.
        (곱 <= 9)동안 {
            정수 결과는 단 * 곱.
            "%d x %d = %d\n"을 단을 곱을 결과를 쓰다.
            곱을 증가하다.
        }
        "\n"을 쓰다.
        단을 증가하다.
    }
    반환 0.
}
```

### 피보나치 (재귀)

```글
포함 "std.gl"

[정수 수를 피보나치]는 -> 정수 {
    수 <= 1이면 { 수를 반환하다. }
    정수 앞값은 (수 - 1)을 피보나치하다.
    정수 뒷값은 (수 - 2)를 피보나치하다.
    앞값 + 뒷값을 반환하다.
}

[시작하기]는 -> 정수 {
    정수 결과는 10을 피보나치하다.
    "피보나치(10) = {결과}\n"을 쓰다.
    반환 0.
}
```

### 문자열 보간

```글
포함 "std.gl"

[시작하기]는 -> 정수 {
    문자열 이름 = "홍길동".
    정수 나이 = 30.
    문자열 도시 = "서울".
    정수 인원 = 5.

    "{이름}은 {도시}에 삽니다\n"을 쓰다.
    "나이: {나이}세, 팀원: {인원}명\n"을 쓰다.
    반환 0.
}
```

## 아키텍처

```
.글 소스 → 형태소분석 → 구문분석 → AST → IR
                                        ↓
                              ┌─────────┴─────────┐
                              ↓                   ↓
                         C 코드 생성          x86-64 기계어
                              ↓                   ↓
                         cl.exe/gcc           PE 직접 생성
                              ↓                   ↓
                           .exe                 .exe
                       (C 트랜스파일)        (네이티브, 외부도구 0)
```

### 컴파일러 구성

| 컴포넌트 | 파일 | 설명 |
|---------|------|------|
| 형태소분석기 | `자체호스팅/형태소분석기.글` | 한글 접사 분리 (3단계 재귀) |
| 구문분석기 | `자체호스팅/구문분석기.글` | SOV 어순 재귀 하강 파서 |
| IR 변환기 | `자체호스팅/IR변환기.글` | AST → 3주소 코드 |
| C 코드생성기 | `자체호스팅/코드생성기.글` | IR → C11 코드 |
| PE 생성기 | `자체호스팅/PE생성기.글` | IR → x86-64 → PE 직접 |
| 표준 라이브러리 | `표준/*.글` | 9개 모듈, 62개 함수 |

### 최적화

- **즉시값 산술** — 상수 즉시값 인코딩
- **비교-분기 융합** — `cmp + jcc` 직접 생성
- **레지스터 핀닝** — 루프 변수를 callee-saved 레지스터에 고정
- **피프홀 최적화** — 불필요한 복사/죽은 저장 제거
- **루프 불변 코드 이동** — 상수 적재를 루프 밖으로
- **꼬리호출 최적화** — 자기 재귀를 `jmp`으로 변환
