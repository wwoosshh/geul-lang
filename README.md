# 글 프로그래밍 언어

**한글로 프로그래밍하는 독자적 프로그래밍 언어입니다.**

```글
[정수 수를 피보나치]는 -> 정수 {
    수 <= 1이면 { 수를 반환하다. }
    정수 앞값은 (수 - 1)을 피보나치하다.
    정수 뒷값은 (수 - 2)를 피보나치하다.
    앞값 + 뒷값을 반환하다.
}

[시작하기]는 {
    정수 결과는 10을 피보나치하다.
    "피보나치(10) = %d\n"을 결과를 쓰다.
}
```

---

## 특징

- **100% 한글 키워드** — 영어 함수명이 사용자 코드에 노출되지 않음
- **SOV 어순** — 한국어 문법처럼 `대상을 함수하다`
- **조사 바인딩** — `을/를`, `에`, `로` 등 조사가 인자 역할 결정
- **네이티브 코드 직접 생성** — C 컴파일러 없이 `.글 → .exe` 직접 컴파일
- **자체호스팅** — 글 컴파일러가 자기 자신을 컴파일
- **VS Code 통합** — 문법 강조, F5 실행, LSP 자동완성

## 성능

> C는 MSVC `/O2` 최적화 적용. 측정: Intel Ultra 5 226V, 5회 중앙값.

| 벤치마크 | 글 네이티브 | C (/O2) | 배율 | Python | 배율 |
|---------|----------:|-------:|-----:|-------:|-----:|
| 재귀 fib(40) | 464 ms | 321 ms | 1.45x | 7,364 ms | 23.0x |
| 소수 체 100만 | 10 ms | 8 ms | 1.30x | 167 ms | 22.0x |
| 버블정렬 3만 | 1,564 ms | 564 ms | 2.77x | 38,377 ms | 68.0x |

**C(/O2) 대비 1.3~2.8x** — Python 대비 22~68x 빠름

## 설치

### 방법 1: 원커맨드 설치 (PowerShell)
```powershell
irm https://raw.githubusercontent.com/wwoosshh/geul-lang/main/install.ps1 | iex
```

### 방법 2: VS Code 확장 수동 설치
1. [릴리즈](https://github.com/wwoosshh/geul-lang/releases)에서 `.vsix` 다운로드
2. VS Code → `Ctrl+Shift+P` → "VSIX에서 설치" → 파일 선택

### 방법 3: 소스에서 빌드
```bash
# 부트스트랩 컴파일러 빌드 (MSVC 필요)
powershell -File 부트스트랩/build_temp.ps1

# 글도구 빌드
부트스트랩/build/글cc.exe 글도구.글

# 네이티브 컴파일러 빌드
부트스트랩/build/글cc.exe 자체호스팅/네이티브컴파일러.글
```

## 사용법

### 글도구 (C 트랜스파일러)
```bash
글도구 실행 소스.글          # 컴파일 + 실행
글도구 빌드 소스.글          # .exe 생성
```

### 네이티브 컴파일러 (C 불필요)
```bash
네이티브컴파일러 소스.글                    # 네이티브 .exe 직접 생성
네이티브컴파일러 소스.글 -출력 결과.exe     # 출력 경로 지정
네이티브컴파일러 주.글 보조.글 -출력 앱.exe  # 다중 파일 빌드
```

### VS Code
| 단축키 | 기능 |
|--------|------|
| **F5** | 컴파일 + 실행 |
| **Ctrl+Shift+B** | 빌드 (.exe 생성) |

## 예제

### 기본 출력
```글
[시작하기]는 {
    "안녕하세요, 글 언어입니다!\n"을 쓰다.
}
```

### 자연스러운 한국어 동사형 호출
```글
[정수 왼쪽을 정수 오른쪽에 더하기]는 -> 정수 {
    왼쪽 + 오른쪽을 반환하다.
}

[시작하기]는 {
    정수 결과는 5를 3에 더하다.
    "5 + 3 = %d\n"을 결과를 쓰다.
}
```

> **동사형 호출**: `쓰다`는 `쓰기` 함수를, `더하다`는 `더하기` 함수를 자동으로 찾습니다. 기존 `쓰다`/`쓰기다` 문법도 여전히 유효합니다.

### 증가/감소 (`증가하다`/`감소하다`)
```글
[시작하기]는 {
    정수 카운터는 0.
    카운터를 증가하다.            (* 카운터++ *)
    카운터를 증가하고 카운터를 증가하다.  (* -고 연결 *)
    "카운터: %d\n"을 카운터를 쓰다.  (* 카운터: 3 *)
}
```

### 구조체
```글
묶음 점은 정수 엑스좌표, 정수 와이좌표.

[시작하기]는 {
    점 위치.
    위치의 엑스좌표 = 10.
    위치의 와이좌표 = 20.
}
```

### 배열 + 동적 메모리
```글
[시작하기]는 {
    문자열 배열은 100을 할당하다.
    배열[0] = 65.
    배열[1] = 66.
    배열을 해제하다.
}
```

### 실수 연산
```글
[시작하기]는 {
    실수 원주율은 3.14159.
    실수 둘레는 원주율 * 2.0 * 5.0.
    "둘레 = %.2f\n"을 둘레를 쓰다.
}
```

## 아키텍처

```
.글 소스 → 형태소분석 → 구문분석 → AST → IR
                                        ↓
                              ┌─────────┴─────────┐
                              ↓                   ↓
                         C 코드 생성          x86-64 기계어
                              ↓                   ↓
                         cl.exe/gcc           PE 직접 생성
                              ↓                   ↓
                           .exe                 .exe
                       (C 트랜스파일)        (네이티브, 외부도구 0)
```

### 컴파일러 구성

| 컴포넌트 | 파일 | 설명 |
|---------|------|------|
| 형태소분석기 | `자체호스팅/형태소분석기.글` | 한글 접사 분리 (3단계 재귀) |
| 구문분석기 | `자체호스팅/구문분석기.글` | SOV 어순 재귀 하강 파서 |
| IR 변환기 | `자체호스팅/IR변환기.글` | AST → 3주소 코드 |
| C 코드생성기 | `자체호스팅/코드생성기.글` | IR → C11 코드 |
| PE 생성기 | `자체호스팅/PE생성기.글` | IR → x86-64 → PE 직접 |
| 표준 라이브러리 | `표준/*.글` | 9개 모듈, 62개 함수 |

### 최적화

- **즉시값 산술** — `add/sub/cmp/imul r10, imm32` 상수 즉시값 인코딩
- **비교-분기 융합** — `cmp + jcc` 직접 생성, setcc/movzx 제거
- **레지스터 핀닝** — 루프 변수를 callee-saved 레지스터(rbx, r12-r15)에 고정
- **피프홀 최적화** — 불필요한 복사/죽은 저장 제거
- **루프 불변 코드 이동** — 상수 적재 및 순수 연산을 루프 밖으로
- **꼬리호출 최적화** — 자기 재귀 꼬리호출을 `jmp`으로 변환
- **결과**: C(/O2) 대비 1.12~2.53x (벤치마크 기준)

## 문법 안내

**[글 언어 문법서 전체 보기](docs/글-언어-문법서.md)**

### 핵심 문법

| 문법 | 글 | C 대응 |
|------|-----|--------|
| 변수 선언 | `정수 값은 42.` | `int 값 = 42;` |
| 함수 정의 | `[정수 수를 제곱]은 -> 정수 { }` | `int 제곱(int 수) { }` |
| 함수 호출 | `42를 제곱하다.` | `제곱(42);` |
| 반환 | `결과를 반환하다.` | `return 결과;` |
| 증가/감소 | `번호를 증가하다.` | `번호++;` |
| 조건문 | `값 > 10이면 { }` | `if (값 > 10) { }` |
| 반복문 | `(조건)동안 { }` | `while (조건) { }` |
| 문장 연결 | `출력하고 증가하다.` | `출력(); 증가();` |
| 구조체 | `묶음 점은 정수 좌표, 정수 와이.` | `struct 점 { int 좌표; int 와이; };` |
| 멤버접근 | `위치의 좌표` | `위치.좌표` |
