# 글 언어 빠른 시작 가이드

## 설치

### 요구 사항
- Windows 10/11
- Visual Studio 2022 (C/C++ 빌드 도구 포함)

### 컴파일러 빌드
```
cd 부트스트랩
빌드.bat
```
빌드 성공 시 `부트스트랩/build/글cc.exe` 생성

## 첫 번째 프로그램

`인사.글` 파일을 만듭니다:

```
[시작하기]는 {
    "안녕하세요, 글 언어입니다!"를 출력하다.
}
```

컴파일 및 실행:
```
글cc.exe 인사.글
인사.exe
```

출력:
```
안녕하세요, 글 언어입니다!
```

## 기본 문법

### 변수 선언
```
정수 나이는 25이다.
실수 키는 175.5이다.
문자열 이름은 "홍길동"이다.
참거짓 활성은 참이다.
```

### 함수 정의
함수는 `[매개변수 함수이름]는 { 본문 }` 형식입니다.

```
[정수 가를 정수 나에 더하기]는 → 정수 {
    반환 가 + 나다.
}
```

- 매개변수 뒤에 조사(`를`, `에`, `에서` 등)를 붙여 역할을 지정
- `→ 타입` 으로 반환 타입 지정
- `반환 값다.` 로 값 반환

### 함수 호출 (SOV 어순)
```
5를 3에 더하기다.          // 더하기(5, 3) → 8
"안녕"을 출력하다.          // 출력("안녕")
```

인자를 먼저 쓰고, 조사를 붙이고, 마지막에 동사(함수 이름)를 씁니다.

### 중첩 호출
괄호로 내부 호출을 먼저 실행합니다:
```
(번호를 피보나치)를 출력하다.   // 출력(피보나치(번호))
```

### 조건문
```
점수가 90보다 크거나같으면 {
    "A등급"을 출력하다.
} 아니면 {
    "B등급 이하"를 출력하다.
}
```

비교 동사: `작은/작으`(<), `큰/크`(>), `작거나같은/작거나같으`(<=), `크거나같은/크거나같으`(>=), `같은/같으`(==), `다른/다르`(!=)

### 반복문
```
정수 번호는 0이다.
번호가 10보다 작은동안 {
    번호를 출력하다.
    번호 = 번호 + 1다.
}
```

### 표준 라이브러리 사용

파일 상단에 `포함` 지시어로 표준 라이브러리를 가져옵니다:

```
포함 "표준/문자열.글"
포함 "표준/메모리.글"

[시작하기]는 {
    문자열 인사는 "안녕하세요"이다.
    정수 길이는 인사를 글_길이다.
    길이를 출력하다.

    공허 참조 메모리는 256을 할당이다.
    메모리를 해제다.
}
```

### 표준 라이브러리 함수

**문자열 (표준/문자열.글)**
| 함수 | 설명 | 사용법 |
|------|------|--------|
| 글_길이 | 문자열 바이트 길이 | `글를 글_길이다` |
| 글_비교 | 두 문자열 비교 | `가를 나와 글_비교다` |
| 글_복사 | 문자열 복사 | `대상에 원본을 글_복사다` |
| 글_연결 | 문자열 연결 | `대상에 원본을 글_연결다` |
| 글_찾기 | 부분문자열 검색 | `건초에서 바늘을 글_찾기다` |
| 정수로_변환 | 문자열→정수 | `글를 정수로_변환다` |

**메모리 (표준/메모리.글)**
| 함수 | 설명 | 사용법 |
|------|------|--------|
| 할당 | 메모리 할당 | `크기를 할당다` |
| 재할당 | 메모리 재할당 | `메모리를 크기로 재할당다` |
| 해제 | 메모리 해제 | `메모리를 해제다` |

**입출력 (표준/입출력.글)**
| 함수 | 설명 | 사용법 |
|------|------|--------|
| 파일_열기 | 파일 열기 | `경로를 모드로 파일_열기다` |
| 파일_닫기 | 파일 닫기 | `파일을 파일_닫기다` |

### 외부 C 함수 사용

C 함수를 직접 호출해야 할 때:
```
외부 [문자열 형식을 ... printf]는 → 정수.

[시작하기]는 {
    "결과: %d\n"를 42를 printf다.
}
```

`외부` 키워드로 C 함수를 선언하고, `...`으로 가변 인자를 지정합니다.
