# 글 컴파일러 내부 구조

## 컴파일 파이프라인

```
.글 소스 → [전처리] → [형태소분석] → [구문분석] → [코드생성] → .c 파일 → [MSVC] → .exe
```

### 1. 전처리기 (주.c: 포함_전처리)
- `포함 "파일경로"` 지시어 처리
- UTF-8 인코딩의 포함 파일을 읽어 소스에 삽입
- 검색 경로: 소스 디렉토리 → 부모 디렉토리 → 컴파일러 디렉토리

### 2. 형태소분석기 (형태소분석기.c)
한글 소스를 토큰 스트림으로 변환합니다.

**핵심 기능:**
- UTF-8 코드포인트 단위 처리
- 한글 단어 → 어근 + 접사(조사/어미) 분리
- 키워드 테이블 대조
- ASCII→한글 경계 분리 (영문 식별자 뒤 한글 조사)
- 밑줄 보호 (복합 식별자 조사 분리 방지)
- 서술격 조사 `이다` 이중분리

**토큰 처리 순서:**
1. 단어 전체가 접사(조사/어미)인지 확인
2. 키워드 테이블에서 검색
3. 뒤에서부터 접사 분리 시도
4. 분리 실패 시 전체를 식별자로 처리

### 3. 구문분석기 (구문분석기.c)
토큰 스트림을 AST(추상 구문 트리)로 변환합니다.

**재귀 하향 파서 구조:**
- `최상위_파싱()` — 전역 선언 (함수, 변수, 묶음, 나열, 외부)
- `문장_파싱()` — 변수선언, 반환, 조건, 반복, 표현식문
- `표현식_파싱()` — 이항 연산 후 SOV 호출 패턴 감지
- `SOV_호출_파싱()` — 한국어 SOV 어순 함수 호출 구성
- `타입_파싱()` — 기본타입, 참조타입, 배열타입

**SOV 파싱 알고리즘:**
```
1. 첫 번째 값과 조사 읽기
2. 반복: 다음 값 파싱 → 조사가 있으면 인자 추가, 없으면 동사(함수이름)
3. 인자 목록과 함수 이름으로 호출 노드 생성
```

### 4. 코드생성기 (코드생성기.c)
AST를 C11 소스코드로 변환합니다.

**주요 변환:**
- `시작하기` → `main`
- 타입 매핑: `정수`→`int`, `문자열`→`char*`, `공허 참조`→`void*` 등
- 비교 동사 → C 비교 연산자 (`작은`→`<`, `큰`→`>` 등)
- 출력 함수 → `_Generic` 매크로 (타입별 자동 포맷)
- C 표준 라이브러리 함수의 extern 재선언 방지

## 파일 구조

```
부트스트랩/
├── include/
│   ├── 구문분석기.h      # 파서 인터페이스
│   ├── 구문트리.h        # AST 노드 정의 (20+ 노드 타입)
│   ├── 유니코드.h        # UTF-8 처리 유틸리티
│   ├── 코드생성기.h      # C 코드 생성기 인터페이스
│   ├── 토큰.h            # 97종 토큰 타입, 조사역할, 어미기능
│   ├── 형태소분석기.h    # 렉서 인터페이스
│   └── llvm생성기.h      # LLVM IR 생성기 (실험적)
├── src/
│   ├── 주.c              # 메인, 전처리기, 파일I/O
│   ├── 형태소분석기.c    # 한글 형태소 분석 렉서
│   ├── 구문분석기.c      # 재귀 하향 파서
│   ├── 코드생성기.c      # C 코드 생성기
│   ├── 구문트리.c        # AST 노드 생성/해제
│   ├── 토큰.c            # 토큰 유틸리티
│   ├── 유니코드.c        # UTF-8 인코딩/디코딩
│   └── llvm생성기.c      # LLVM IR 생성 (실험적)
├── build/
│   └── 글cc.exe          # 컴파일된 컴파일러
├── build.bat             # 빌드 스크립트
├── 빌드.bat              # 빌드 스크립트 (한글)
└── Makefile              # Make 빌드 (gcc용)
```

## AST 노드 타입

| 노드 | 설명 |
|------|------|
| 노드_프로그램 | 최상위 프로그램 |
| 노드_함수정의 | 함수 정의/외부 선언 |
| 노드_변수선언 | 변수 선언 |
| 노드_호출 | 함수 호출 |
| 노드_식별자 | 식별자 참조 |
| 노드_정수_리터럴 | 정수 리터럴 |
| 노드_실수_리터럴 | 실수 리터럴 |
| 노드_문자열_리터럴 | 문자열 리터럴 |
| 노드_이항연산 | 이항 연산 (+, -, *, / 등) |
| 노드_단항연산 | 단항 연산 (-, !) |
| 노드_조건문 | if/else |
| 노드_동안반복 | while |
| 노드_블록 | 코드 블록 {} |
| 노드_반환문 | return |
| 노드_대입 | 대입 |
| 노드_기본타입 | 기본 타입 노드 |
| 노드_참조타입 | 포인터 타입 |
| 노드_배열타입 | 배열 타입 |
| 노드_묶음정의 | struct 정의 |
| 노드_나열정의 | enum 정의 |
| 노드_멤버접근 | struct.field |
| 노드_배열접근 | array[index] |
| 노드_형변환 | 타입 캐스트 |
| 노드_크기연산 | sizeof |
