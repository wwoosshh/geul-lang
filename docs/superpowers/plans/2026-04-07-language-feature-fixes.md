## 다음 작업 권장 순서

**즉시 (쉬운 승리):**
1. **B2 static** — BSS 섹션에 정적 변수 배치 (~30줄)
2. **D4 dead import 제거** — 사용된 함수만 임포트 (~50줄)
3. **C6 한글 어미 확장** — 파서에서 `-서/-지만` 등 처리 (~30줄)

**단기 (높은 임팩트):**
4. **C1 goto/label** — 파서+IR+코드젠 (~150줄)
5. **C2 range-for** — 파서 디슈가링 (~50줄)
6. **C5 에러 처리** — 설계 문서 기반 구현 (~200줄)

**중기:**
7. **A4 va_list** — Windows x64 ABI (~200줄)
8. **B1 구조체 필드 크기** — 타입별 크기 계산 + 정렬 (~300줄)
9. **B4/B5 LICM/인라이닝** — 기존 버그 디버깅 + 활성화

**장기:**
10. **D1 다중 파일** → **D2 디버그** → **D5 ELF** → **D8 CRT 제거**
