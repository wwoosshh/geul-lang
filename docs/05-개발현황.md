# 글 언어 개발 현황

## 현재 버전: 0.2 (자체호스팅)

### 개발 단계

| 단계 | 설명 | 상태 |
|------|------|------|
| Phase 0 | C로 부트스트랩 컴파일러 구현 | **완료** |
| Phase 1 | 기본 컴파일러 동작 (변수, 함수, 제어흐름) | **완료** |
| Phase 2 | 확장 기능 (포함, 표준 라이브러리, 구조체, 배열 등) | **완료** |
| Phase 3 | 글 언어로 컴파일러 재작성 (자체 호스팅) | **완료** (2026-03-17) |
| Phase 4 | 한글 전용화 + 표준 라이브러리 확장 | **완료** (2026-03-17) |
| Phase 5 | 네이티브 백엔드 (C 독립) | 미시작 |

### 자체호스팅 달성 (2026-03-17)

- 컴파일러 소스 100% 글 언어 (7개 파일, ~4100줄)
- 고정점 도달: Stage1 == Stage2 SHA256 일치
- SHA256: `0C122A963AD3E951E919D9B1D0687430E1B93511506C25B23874BA0FC96385A9`
- 빌드: 부트스트랩 → Stage0 → Stage1 → Stage2 (바이트 동일)

### 완성된 기능

#### 기본 기능
- **함수 정의/호출** — SOV(주어-목적어-동사) 어순 파싱
- **조사 기반 인자 바인딩** — 를/을(목적어), 에(목적지), 에서(출처), 보다(비교), 와/과(동반), 로(수단)
- **어미 기반 제어흐름** — 다(서술), 면(조건/if), 동안(반복/while)
- **타입 시스템** — 정수, 긴정수, 짧은정수, 실수, 짧은실수, 문자, 문자열, 참거짓, 공허
- **포인터** — `참조` 키워드 (다중 참조: `공허 참조 참조` → `void**`)
- **산술/비교 연산** — +, -, *, /, %, 작은/큰/같은/다른 등
- **재귀** — 피보나치 등 재귀 함수 정상 동작

#### 제어흐름
- **조건문** — `-면` / `아니면` / else-if 체이닝
- **while 반복** — `-동안`
- **범위반복** — `부터/까지`
- **C-style for** — `반복 (초기화: 조건: 증감) { }`
- **do-while** — `반복하기 { } 조건 동안.`
- **갈래/경우/기본** — switch/case/default
- **탈출(break)**, **계속(continue)**

#### 타입 및 선언
- **묶음(struct)** — 필드 선언, 인스턴스 초기화 `{ 필드: 값, ... }`
- **나열(enum)** — 열거형 정의
- **합침(union)** — `합침 이름은 필드목록.`
- **별칭(typedef)** — `별칭 새이름은 기존타입.`
- **배열** — 선언/초기화/접근, 배열 리터럴 (`정수[50] 이름.`)
- **함수 포인터** — `[타입, 타입 → 반환타입]` 완전 구현

#### 연산자 및 기타
- **복합 대입** — +=, -=, *=, /=, %=
- **증감 연산자** — 전위/후위 ++, --
- **삼항 연산자** — `조건 ? 참값 : 거짓값`
- **형변환** — `값을 정수로` → `(int)(값)` 캐스팅
- **sizeof** — `크기` 키워드
- **멤버 접근** — `구조체의 필드`, `포인터→필드`

#### 전처리 및 모듈
- **포함 시스템** — `포함 "파일경로"` 지시어, 재귀 포함, 중복 방지
- **외부 함수 선언** — `외부 [타입 이름을 C함수명]는 → 반환타입.`
- **조건부 컴파일** — `만약정의/만약미정의/아니면/끝` (#ifdef/#ifndef/#else/#endif)
- **정의(#define)** — 매크로 정의
- **정적(static)** — 함수/변수에 `정적` 접두사
- **상수(const)** — 변수/매개변수에 `상수` 접두사
- **가변인자** — 가변목록(va_list), 가변시작/가변끝/가변인자
- **직접 호출 구문** — `함수이름()` 또는 `함수이름(인자들)` C 스타일

#### 표준 라이브러리 (한글 래퍼) — 7개 모듈
- **입출력** — 쓰기, 파일_열기, 파일_닫기, 파일_읽기, 파일_쓰기, 출력_전환, 파일_삭제 등
- **문자열** — 글자수, 문자열_비교, 문자열_같은가, 문자열_복제, 문자열_일부비교 등
- **메모리** — 할당, 해제, 재할당, 초기할당, 메모리_복사, 메모리_채우기
- **체계** — 종료, 명령_실행, 절댓값
- **수학** — 제곱근, 거듭제곱, 반올림, 올림, 내림, 최솟값, 최댓값, 삼각함수 등
- **변환** — 글을_정수로, 글을_실수로, 정수를_글로, 실수를_글로
- **시간** — 현재시각, 시간차이, 경과틱
- **기본** — 통합 모듈 (`포함 "표준/기본.글"` 한 줄로 기본 기능 전부)

#### 형태소 분석
- **재귀 접사 분리** — `정수로다` → `정수` + `로` + `다`
- **서술격 조사 2.5단계** — `할당이면` → `할당` + `이` + `면`
- **분리제외 사전** — 복합어/외래어 보호

### 미완성 기능

- 다중 파일 독립 컴파일 (헤더 생성 등)
- 문자열 보간
- 동적 배열/리스트

### 알려진 제한사항

- Windows 전용 (MSVC 의존)
- C 트랜스파일 방식 (`.글 → .c → .exe`)
- 밑줄 포함 함수명(`파일_열기` 등)은 SOV 호출 불가 — 직접호출 구문 사용 필수
