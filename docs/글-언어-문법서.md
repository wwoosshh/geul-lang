## 1. 시작하기

### 첫 프로그램

```글
[시작하기]는 {
    "안녕하세요! 글 언어입니다.\n"을 쓰기다.
}
```

- `[시작하기]는 { }` — 프로그램의 시작점 (C의 `main()`)
- `"..."을 쓰기다.` — 화면에 출력 (C의 `printf()`)
- 모든 문장은 `.`으로 끝남

### 실행 방법

VS Code에서 `.글` 파일을 열고 **F5**를 누르면 실행됩니다.

## 2. 데이터 타입

| 타입 | 설명 | 예시 |
|------|------|------|
| `정수` | 32비트 정수 | `정수 나이 = 25.` |
| `긴정수` | 64비트 정수 | `긴정수 큰수 = 9999999999.` |
| `실수` | 64비트 소수 | `실수 원주율 = 3.14.` |
| `문자` | 1바이트 글자 | `문자 기호 = 'A'.` |
| `문자열` | 문자열 | `문자열 이름 = "홍길동".` |
| `참거짓` | 참/거짓 | `참거짓 활성 = 참.` |
| `공허` | 반환값 없음 | 함수 반환 타입 |

### 실습

```글
[시작하기]는 {
    정수 나이 = 25.
    실수 키 = 175.5.
    문자열 이름 = "홍길동".
    참거짓 학생 = 참.

    "이름: %s, 나이: %d\n"을 이름을 나이를 쓰기다.
}
```

## 4. 연산자

### 산술

```글
[시작하기]는 {
    정수 결과 = 0.
    결과 = 10 + 3.       (* 13 — 더하기 *)
    결과 = 10 - 3.       (* 7  — 빼기 *)
    결과 = 10 * 3.       (* 30 — 곱하기 *)
    결과 = 10 / 3.       (* 3  — 나누기 *)
    결과 = 10 % 3.       (* 1  — 나머지 *)
}
```

### 비교

```글
10 == 10    (* 참 — 같다 *)
10 != 5     (* 참 — 다르다 *)
10 > 5      (* 참 — 크다 *)
10 < 5      (* 거짓 — 작다 *)
10 >= 10    (* 참 — 크거나같다 *)
10 <= 5     (* 거짓 — 작거나같다 *)
```

### 논리

```글
참 && 참      (* 참 — 그리고 *)
참 || 거짓    (* 참 — 또는 *)
!참           (* 거짓 — 부정 *)
```

### 대입

```글
값 = 10.       (* 대입 *)
값 += 5.       (* 값 = 값 + 5 *)
값 -= 3.       (* 값 = 값 - 3 *)
값 *= 2.       (* 값 = 값 * 2 *)
값 /= 4.       (* 값 = 값 / 4 *)
값 %= 3.       (* 값 = 값 % 3 *)
```

### 증감

```글
값++다.         (* 후위 증가: 값을 쓰고 +1 *)
++값.           (* 전위 증가: +1 하고 값을 씀 *)
값--다.         (* 후위 감소 *)

(* 자연어 스타일 증감 *)
값을 증가하다.   (* 값++ — 한국어 동사형 *)
값을 감소하다.   (* 값-- *)
```

### 삼항

```글
정수 결과 = 나이 >= 20 ? 1 : 0.
문자열 등급 = 점수 >= 90 ? "A" : "B".
```

## 5. 함수

### 함수 정의

```글
[정수 왼쪽을 정수 오른쪽을 더하기]는 -> 정수 {
    왼쪽 + 오른쪽을 반환하다.
}
```

**구조**: `[매개변수들 함수이름]은/는 -> 반환타입 { 본문 }`

- 매개변수: `타입 이름+조사` 형태
- `->` 뒤에 반환 타입 (없으면 생략)
- `값을 반환하다.` 또는 `반환 값.` 으로 값 반환

### 함수 호출

**방법 1 — SOV 호출** (자연스러운 한국어):
```글
정수 결과는 10을 20을 더하기하다.
```

**방법 2 — 간결한 SOV 호출** (하다 생략):
```글
정수 결과는 10을 20을 더하기다.
```

**방법 3 — 직접 호출** (C 스타일):
```글
정수 결과 = 더하기(10, 20).
```

> **참고**: `함수이름하다`와 `함수이름다` 모두 유효합니다. `하다` 형태가 더 자연스러운 한국어이며, `다` 형태는 간결한 대안입니다.

### 반환문

```글
(* 방법 1 — 자연어 스타일 (SOV) *)
결과를 반환하다.

(* 방법 2 — 키워드 스타일 *)
반환 결과.

(* 방법 3 — 값 없는 반환 *)
반환.
```

### 반환값이 없는 함수

```글
[문자열 메시지를 인사하기]는 {
    "인사: %s\n"을 메시지를 쓰기하다.
}

[시작하기]는 {
    "안녕하세요"를 인사하기하다.
}
```

### 재귀 함수

```글
[정수 숫자를 팩토리얼]은 -> 정수 {
    숫자 <= 1이면 {
        1을 반환하다.
    }
    숫자 * 팩토리얼(숫자 - 1)을 반환하다.
}

[시작하기]는 {
    "10! = %d\n"을 팩토리얼(10)을 쓰기하다.
}
```

## 6. 조사와 어미

### 조사 — 인자의 역할 지정

글 언어의 가장 큰 특징입니다. 조사가 인자의 역할을 결정합니다.

| 조사 | 역할 | 예시 |
|------|------|------|
| `을/를` | 목적어 (대상) | `"안녕"을 쓰기다.` |
| `에` | 목적지 | `파일에 쓰다.` |
| `에서` | 출처 | `파일에서 읽다.` |
| `로/으로` | 수단/변환 | `정수로 변환하다.` |
| `와/과` | 동반/비교 | `값1과 비교하다.` |
| `의` | 소유/멤버 | `학생의 이름` |

> **받침 규칙**: 받침 있으면 `을/은/이/과/으로`, 없으면 `를/는/가/와/로`

### 어미 — 문장의 역할 지정

| 어미 | 의미 | 예시 |
|------|------|------|
| `-다` | 문장 종결 | `출력하다.` |
| `-면` | 조건 (if) | `크면 { }` |
| `-동안` | 반복 (while) | `작은동안 { }` |
| `-고` | 문장 연결 (순차) | `출력하고 증가하다.` |

### `-고` 문장 연결

한국어의 연결 어미 `-고`를 사용하여 여러 문장을 하나로 연결할 수 있습니다.

```글
(* 두 동작을 순차적으로 실행 *)
"결과: %d\n"을 값을 쓰기하고 값을 증가하다.

(* 위와 동일한 동작 *)
"결과: %d\n"을 값을 쓰기하다.
값을 증가하다.
```

### 실습 — 조사로 인자 순서 자유롭게

```글
[정수 물건값을 정수 할인율로 할인계산]은 -> 정수 {
    반환 물건값 - (물건값 * 할인율 / 100).
}

[시작하기]는 {
    (* 어떤 순서든 조사가 역할을 결정 *)
    정수 결과1 = 10000을 20으로 할인계산다.
    정수 결과2 = 20으로 10000을 할인계산다.
    (* 결과1 == 결과2 == 8000 *)
}
```

## 7. 제어문

### 조건문 (-면 / 아니면)

```글
[시작하기]는 {
    정수 점수 = 85.

    점수 >= 90이면 {
        "A등급\n"을 쓰기다.
    }
    아니면 점수 >= 80이면 {
        "B등급\n"을 쓰기다.
    }
    아니면 점수 >= 70이면 {
        "C등급\n"을 쓰기다.
    }
    아니면 {
        "F등급\n"을 쓰기다.
    }
}
```

### while 반복 (-동안)

```글
[시작하기]는 {
    정수 합계 = 0.
    정수 번호 = 1.

    (번호 <= 100)동안 {
        합계 += 번호.
        번호++다.
    }

    "1~100 합: %d\n"을 합계를 쓰기다.
}
```

### for 반복 (반복)

```글
[시작하기]는 {
    반복 (정수 번호 = 1: 번호 <= 10: 번호++) {
        "%d "를 번호를 쓰기다.
    }
    "\n"을 쓰기다.
}
```

> **주의**: 구분자는 세미콜론(`;`)이 아니라 쌍점(`:`)

### do-while (반복하기)

```글
[시작하기]는 {
    정수 수 = 1.
    반복하기 {
        "%d\n"을 수를 쓰기다.
        수++다.
    } 수 <= 5 동안.
}
```

### switch (갈래/경우/기본)

```글
[시작하기]는 {
    정수 요일 = 3.

    갈래 요일 {
        경우 1: { "월요일\n"을 쓰기다. }
        경우 2: { "화요일\n"을 쓰기다. }
        경우 3: { "수요일\n"을 쓰기다. }
        기본: { "기타\n"을 쓰기다. }
    }
}
```

### 탈출과 계속

```글
[시작하기]는 {
    반복 (정수 번호 = 0: 번호 < 100: 번호++) {
        번호 == 50이면 { 탈출. }      (* break — 반복 종료 *)
        (번호 % 2) == 0이면 { 계속. } (* continue — 다음 반복 *)
        "%d "를 번호를 쓰기다.
    }
}
```

## 8. 배열

### 선언

```글
정수[5] 점수.                          (* 크기 5 배열 *)
정수[3] 값들 = { 10, 20, 30 }.        (* 초기화 *)
문자[256] 버퍼.                        (* 문자 버퍼 *)
```

> **문법**: `타입[크기] 이름` (C의 `타입 이름[크기]`와 다름)

### 사용

```글
[시작하기]는 {
    정수[5] 점수 = { 90, 85, 78, 92, 88 }.

    정수 합계 = 0.
    반복 (정수 번호 = 0: 번호 < 5: 번호++) {
        합계 += 점수[번호].
    }

    "평균: %d\n"을 (합계 / 5)를 쓰기다.
}
```

## 11. 포인터와 참조

### 포인터 선언

```글
정수 값 = 42.
정수 참조 포인터 = &값.     (* 주소 가져오기 *)
```

> `타입 참조` = C의 `타입*`

### 역참조

```글
*포인터 = 100.              (* 포인터가 가리키는 값 변경 *)
```

### 화살표 접근 (->)

```글
학생 참조 포인터 = &철수.
포인터→이름을 쓰기다.       (* 포인터->이름 *)
포인터→점수 = 100.
```

### 동적 메모리

```글
[시작하기]는 {
    정수 참조 배열 = 할당(100 * 크기(정수)).

    반복 (정수 번호 = 0: 번호 < 100: 번호++) {
        배열[번호] = 번호 * 2.
    }

    배열을 해제다.
}
```

## 12. 표준 라이브러리

표준 라이브러리는 자동으로 포함됩니다. 별도 `포함` 없이 바로 사용 가능합니다.

### 출력 (쓰기)

```글
"안녕하세요\n"을 쓰기다.                     (* 문자열 출력 *)
"이름: %s, 나이: %d\n"을 이름을 나이를 쓰기다. (* 형식 출력 *)
```

형식 지정자:
- `%d` — 정수
- `%lld` — 긴정수
- `%f` — 실수
- `%s` — 문자열
- `%c` — 문자
- `%x` — 16진수
- `%%` — % 기호 자체

### 문자열 함수

```글
정수 길이 = 글자수("안녕하세요").        (* strlen — 문자열 길이 *)
정수 같은가 = 문자열_비교("가", "나").    (* strcmp — 비교 *)
문자열 복사본 = 문자열_복제("원본").      (* strdup — 복제 *)
정수 숫자 = 정수로_변환("42").           (* atoi — 문자열→정수 *)
```

### 메모리 함수

```글
공허 참조 메모리 = 할당(1024).           (* malloc *)
메모리를 해제다.                          (* free *)
공허 참조 깨끗한 = 초기할당(10, 4).      (* calloc *)
메모리 = 메모리를 2048로 재할당다.        (* realloc *)
```

### 파일 입출력

```글
공허 참조 파일 = 파일_열기("데이터.txt", "r").
문자[256] 줄.
줄_읽기(줄, 256, 파일).
"%s"를 줄을 쓰기다.
파일_닫기(파일).
```

### 시스템

```글
"dir"을 명령_실행다.          (* system() — 시스템 명령 실행 *)
정수 절대 = 절댓값(-42).     (* abs() — 절댓값 *)
0으로 종료다.                 (* exit() — 프로그램 종료 *)
```

## 14. 고급 기능

### 별칭 (typedef)

```글
별칭 새이름은 기존타입.
별칭 콜백은 [정수, 정수 -> 정수].
```

### 합침 (union)

```글
합침 데이터는
    정수 숫자,
    실수 소수,
    문자 글자.
```

### 함수 포인터

```글
[정수 왼쪽을 정수 오른쪽을 더하기]는 -> 정수 { 반환 왼쪽 + 오른쪽. }
[정수 왼쪽을 정수 오른쪽을 빼기]는 -> 정수 { 반환 왼쪽 - 오른쪽. }

[시작하기]는 {
    [정수, 정수 -> 정수] 연산 = 더하기.
    "%d\n"을 연산(10, 3)을 쓰기다.    (* 13 *)

    연산 = 빼기.
    "%d\n"을 연산(10, 3)을 쓰기다.    (* 7 *)
}
```

### 외부 함수 선언

C 라이브러리 함수를 직접 사용할 때:

```글
외부 [문자열 형식을 ... printf]는 -> 정수.
외부 [문자열 형식을 ... scanf]는 -> 정수.
```

### 가변 인자

```글
[정수 개수를 ... 모두더하기]는 -> 정수 {
    가변목록 목록.
    가변시작(목록, 개수).
    정수 합계 = 0.
    반복 (정수 번호 = 0: 번호 < 개수: 번호++) {
        합계 += 가변인자(목록, 정수).
    }
    가변끝(목록).
    반환 합계.
}

[시작하기]는 {
    정수 결과 = 모두더하기(3, 10, 20, 30).
    "합: %d\n"을 결과를 쓰기다.    (* 합: 60 *)
}
```
