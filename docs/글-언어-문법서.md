# 글 프로그래밍 언어 문법서

> 한글로 프로그래밍하는 언어, **글(Geul)**의 완전한 문법 안내서입니다.
>
> 글은 한국어의 교착어 특성을 살려, **조사**가 인자의 역할을, **어미**가 제어 흐름을 결정하는 독자적 프로그래밍 언어입니다. `.글` 소스 파일을 C 또는 네이티브 x86-64 기계어로 컴파일합니다.

## 1. 기본 구조

### 첫 프로그램

```글
[시작하기]는 {
    "안녕하세요! 글 언어입니다.\n"을 쓰기하다.
}
```

- `[시작하기]는 { }` — 프로그램의 시작점 (C의 `main()`)
- `"..."을 쓰기하다.` — 화면에 출력 (C의 `printf()`)
- 모든 문장은 `.`으로 끝남
- 주석은 `(* 이것은 주석입니다 *)` 형태

### 문장의 기본 규칙

글 언어의 모든 문장은 **마침표(`.`)**로 끝납니다. 한국어 문장이 마침표로 끝나듯이, 글 프로그램도 마침표로 문장을 구분합니다.

```글
"안녕\n"을 쓰기하다.         (* 출력 문장 *)
정수 값 = 42.               (* 변수 선언 *)
값 = 100.                   (* 대입 문장 *)
```

### 실행 방법

```
글도구 실행 프로그램.글          컴파일 후 즉시 실행
글도구 빌드 프로그램.글          실행파일(.exe) 생성
글도구 검사 프로그램.글          문법 검사만
```

VS Code에서 `.글` 파일을 열고 **F5**를 누르면 바로 실행됩니다.

## 2. 타입과 변수

### 기본 타입

| 타입 | 설명 | 예시 |
|------|------|------|
| `정수` | 32비트 정수 | `정수 나이 = 25.` |
| `긴정수` | 64비트 정수 | `긴정수 큰수 = 9999999999.` |
| `실수` | 64비트 소수 | `실수 원주율 = 3.14.` |
| `문자` | 1바이트 글자 | `문자 기호 = 'A'.` |
| `문자열` | 문자열 | `문자열 이름 = "홍길동".` |
| `참거짓` | 참/거짓 | `참거짓 활성 = 참.` |
| `공허` | 반환값 없음 | 함수 반환 타입 |

### 변수 선언

#### 타입 명시 선언

```글
정수 점수 = 100.           (* 선언 + 초기화 *)
정수 합계.                  (* 선언만 *)
합계 = 0.                   (* 나중에 대입 *)
```

#### 타입 추론 선언

초기값이 있으면 타입을 생략할 수 있습니다. 컴파일러가 리터럴의 종류에서 타입을 자동으로 추론합니다.

```글
값은 42.                    (* 정수로 추론 *)
원주율은 3.14.              (* 실수로 추론 *)
이름은 "철수".              (* 문자열로 추론 *)
완료는 참.                  (* 참거짓으로 추론 *)

정수 값은 42.               (* 타입 명시도 여전히 유효 *)
```

**추론 규칙:**

| 초기값 형태 | 추론 타입 |
|------------|----------|
| 정수 리터럴 (`42`) | `정수` |
| 실수 리터럴 (`3.14`) | `실수` |
| 문자열 리터럴 (`"..."`) | `문자열` |
| 문자 리터럴 (`'가'`) | `문자` |
| `참` / `거짓` | `참거짓` |

> **주의**: 포인터, 배열, 묶음(구조체) 등 복합 타입은 추론되지 않습니다. 반드시 타입을 명시하세요.

### 상수

```글
상수 정수 최대값 = 1000.    (* 변경 불가 *)
상수 문자열 제목 = "계산기".
```

### 정적 변수

```글
정적 정수 호출횟수 = 0.     (* 함수 호출 간 값 유지 *)
```

## 3. 함수

### 함수 정의

```글
[정수 왼쪽을 정수 오른쪽을 더하기]는 -> 정수 {
    왼쪽 + 오른쪽을 반환하다.
}
```

**구조**: `[매개변수들 함수이름]은/는 -> 반환타입 { 본문 }`

- 매개변수: `타입 이름+조사` 형태로, 조사가 인자의 역할을 결정
- `->` 뒤에 반환 타입 (반환값이 없으면 생략)
- 함수 이름은 대괄호의 마지막 단어

### 함수 호출

글 언어에서 함수를 호출하는 방법은 세 가지입니다.

**방법 1 — `하다` 호출** (가장 자연스러운 한국어):
```글
정수 결과는 10을 20을 더하기하다.
```

`하다`는 한국어에서 명사를 동사로 만드는 보편적인 접미사입니다. `더하기하다`는 "더하기를 하다"와 같은 뜻으로, 가장 자연스러운 호출 형태입니다.

**방법 2 — 간결한 호출** (`하다` 생략):
```글
정수 결과는 10을 20을 더하기다.
```

**방법 3 — 직접 호출** (C 스타일 괄호):
```글
정수 결과 = 더하기(10, 20).
```

> **`하다` 동사 활용**: `하다`는 렉서에서 투명하게 제거됩니다. `계산하다`, `출력하고`, `처리하면` 등 모든 어미 활용에서 `하`는 자동으로 분리됩니다. 따라서 `쓰기다`와 `쓰기하다`는 완전히 동일하게 동작합니다.

### `하다` 활용 예시

```글
(* 모두 같은 동작 *)
"안녕\n"을 쓰기하다.        (* 하다 — 가장 자연스러운 형태 *)
"안녕\n"을 쓰기다.          (* 간결한 형태 *)
쓰기("안녕\n").             (* 직접 호출 *)

(* 어미가 바뀌어도 하 분리가 적용됨 *)
값을 계산하고 결과를 출력하다.     (* 하고 → 하+고, 하다 → 하+다 *)
```

### 반환문

반환문도 자연어 스타일과 키워드 스타일 두 가지를 지원합니다.

```글
(* 방법 1 — SOV 자연어 스타일 (권장) *)
결과를 반환하다.

(* 방법 2 — 키워드 스타일 *)
반환 결과.

(* 방법 3 — 값 없는 반환 *)
반환.
```

SOV 반환은 파서가 동사 위치에서 `반환` 키워드를 인식하여 반환문으로 처리합니다. `결과를 반환하다.`는 "결과를 반환한다"라는 자연스러운 한국어 문장입니다.

### 반환값이 없는 함수

```글
[문자열 메시지를 인사하기]는 {
    "인사: %s\n"을 메시지를 쓰기하다.
}

[시작하기]는 {
    "안녕하세요"를 인사하기하다.
}
```

### 재귀 함수

```글
[정수 숫자를 팩토리얼]은 -> 정수 {
    숫자 <= 1이면 {
        1을 반환하다.
    }
    숫자 * 팩토리얼(숫자 - 1)을 반환하다.
}

[시작하기]는 {
    "10! = %d\n"을 팩토리얼(10)을 쓰기하다.
}
```

## 4. 제어 흐름

### 조사와 어미 — 글 언어의 핵심

글 언어의 가장 큰 특징은 한국어의 **조사**와 **어미**를 프로그래밍에 활용하는 것입니다.

#### 조사 — 인자의 역할 지정

| 조사 | 역할 | 예시 |
|------|------|------|
| `을/를` | 목적어 (대상) | `"안녕"을 쓰기하다.` |
| `에` | 목적지 | `파일에 쓰다.` |
| `에서` | 출처 | `파일에서 읽다.` |
| `로/으로` | 수단/변환 | `정수로 변환하다.` |
| `와/과` | 동반/비교 | `값1과 비교하다.` |
| `의` | 소유/멤버 | `학생의 이름` |

> **받침 규칙**: 받침 있으면 `을/은/이/과/으로`, 없으면 `를/는/가/와/로`

#### 어미 — 문장의 역할 지정

| 어미 | 의미 | 예시 |
|------|------|------|
| `-다` | 문장 종결 | `출력하다.` |
| `-면` | 조건 (if) | `크면 { }` |
| `-동안` | 반복 (while) | `작은동안 { }` |
| `-고` | 문장 연결 (순차) | `출력하고 증가하다.` |

### 조건문 (-면 / 아니면)

```글
[시작하기]는 {
    정수 점수 = 85.

    점수 >= 90이면 {
        "A등급\n"을 쓰기하다.
    }
    아니면 점수 >= 80이면 {
        "B등급\n"을 쓰기하다.
    }
    아니면 점수 >= 70이면 {
        "C등급\n"을 쓰기하다.
    }
    아니면 {
        "F등급\n"을 쓰기하다.
    }
}
```

### 없으면 / 있으면 — 자연어 널 검사

포인터나 값이 0(없음/널)인지 확인할 때, 자연스러운 한국어 표현을 사용할 수 있습니다.

```글
공허 참조 파일 = 파일_열기("데이터.txt", "r").

(* 자연어 스타일 — 권장 *)
파일이 없으면 {
    "파일을 열 수 없습니다\n"을 쓰기하다.
    1로 종료하다.
}

파일이 있으면 {
    "파일 열기 성공\n"을 쓰기하다.
}

(* 동일한 의미의 비교 연산 스타일 *)
파일 == 0이면 { (* ... *) }
파일 != 0이면 { (* ... *) }
```

- `없으면` = `== 0이면` (널/영 검사)
- `있으면` = `!= 0이면` (비널/비영 검사)

### while 반복 (-동안)

```글
[시작하기]는 {
    합계는 0.
    번호는 1.

    (번호 <= 100)동안 {
        합계 += 번호.
        번호를 증가하다.
    }

    "1~100 합: %d\n"을 합계를 쓰기하다.
}
```

자연어 비교 동사도 사용할 수 있습니다:

```글
번호가 100보다 작거나같은동안 {
    (* 본문 *)
}
```

### for 반복 (반복)

```글
[시작하기]는 {
    반복 (정수 번호 = 1: 번호 <= 10: 번호++) {
        "%d "를 번호를 쓰기하다.
    }
    "\n"을 쓰기하다.
}
```

> **주의**: 구분자는 세미콜론(`;`)이 아니라 **쌍점(`:`)**입니다.

### do-while (반복하기)

```글
[시작하기]는 {
    정수 수 = 1.
    반복하기 {
        "%d\n"을 수를 쓰기하다.
        수를 증가하다.
    } 수 <= 5 동안.
}
```

### switch (갈래/경우/기본)

```글
[시작하기]는 {
    정수 요일 = 3.

    갈래 요일 {
        경우 1: { "월요일\n"을 쓰기하다. }
        경우 2: { "화요일\n"을 쓰기하다. }
        경우 3: { "수요일\n"을 쓰기하다. }
        기본: { "기타\n"을 쓰기하다. }
    }
}
```

### 탈출과 계속

```글
[시작하기]는 {
    반복 (정수 번호 = 0: 번호 < 100: 번호++) {
        번호 == 50이면 { 탈출. }      (* break — 반복 종료 *)
        (번호 % 2) == 0이면 { 계속. } (* continue — 다음 반복 *)
        "%d "를 번호를 쓰기하다.
    }
}
```

### `-고` 문장 연결

한국어의 연결 어미 `-고`를 사용하여 여러 동작을 한 문장으로 이어 쓸 수 있습니다. 순차적으로 실행됩니다.

```글
(* -고로 두 동작을 순차 실행 *)
"결과: %d\n"을 값을 쓰기하고 값을 증가하다.

(* 위와 동일한 동작을 두 문장으로 쓴 것 *)
"결과: %d\n"을 값을 쓰기하다.
값을 증가하다.
```

`-고`는 어미이므로 `하다`, `다` 어간 뒤에 자연스럽게 붙습니다:

```글
파일을 열하고 내용을 읽하고 파일을 닫하다.    (* 세 동작 연쇄 *)
```

## 5. 연산자

### 산술

```글
[시작하기]는 {
    결과는 0.
    결과 = 10 + 3.       (* 13 — 더하기 *)
    결과 = 10 - 3.       (* 7  — 빼기 *)
    결과 = 10 * 3.       (* 30 — 곱하기 *)
    결과 = 10 / 3.       (* 3  — 나누기 *)
    결과 = 10 % 3.       (* 1  — 나머지 *)
}
```

### 비교

```글
10 == 10    (* 참 — 같다 *)
10 != 5     (* 참 — 다르다 *)
10 > 5      (* 참 — 크다 *)
10 < 5      (* 거짓 — 작다 *)
10 >= 10    (* 참 — 크거나같다 *)
10 <= 5     (* 거짓 — 작거나같다 *)
```

자연어 비교 동사도 사용할 수 있습니다:

```글
점수가 90보다 크면 { (* ... *) }
번호가 10보다 작은동안 { (* ... *) }
값이 기준과 같으면 { (* ... *) }
번호가 100보다 작거나같은동안 { (* ... *) }
```

### 논리

```글
참 && 참      (* 참 — 그리고 *)
참 || 거짓    (* 참 — 또는 *)
!참           (* 거짓 — 부정 *)
```

### 대입

```글
값 = 10.       (* 대입 *)
값 += 5.       (* 값 = 값 + 5 *)
값 -= 3.       (* 값 = 값 - 3 *)
값 *= 2.       (* 값 = 값 * 2 *)
값 /= 4.       (* 값 = 값 / 4 *)
값 %= 3.       (* 값 = 값 % 3 *)
```

### 증감

```글
(* 자연어 동사형 증감 — 권장 *)
값을 증가하다.    (* 값++ *)
값을 감소하다.    (* 값-- *)

(* 기호 스타일 *)
값++다.           (* 후위 증가: 값을 쓰고 +1 *)
++값.             (* 전위 증가: +1 하고 값을 씀 *)
값--다.           (* 후위 감소 *)
--값.             (* 전위 감소 *)
```

`증가하다`와 `감소하다`는 파서가 SOV 동사 위치에서 인식하는 내장 동사입니다. `을/를` 조사로 대상을 지정합니다.

```글
(* -고 연결과 조합하여 *)
값을 출력하고 값을 증가하다.       (* 출력 후 증가 *)
번호를 증가하고 합계에 번호를 더하다.
```

### 삼항

```글
정수 결과 = 나이 >= 20 ? 1 : 0.
문자열 등급 = 점수 >= 90 ? "A" : "B".
```

## 6. 구조체와 배열

### 구조체 (묶음)

#### 정의

```글
묶음 학생은
    문자열 이름,
    정수 나이,
    정수 점수.
```

> **문법**: 필드는 쉼표로 구분, 마지막에 마침표. 중괄호(`{}`)를 쓰지 않습니다.

#### 인스턴스 생성

```글
[시작하기]는 {
    학생 철수 = { 이름: "김철수", 나이: 20, 점수: 95 }.
    "이름: %s, 점수: %d\n"을 철수.이름을 철수.점수를 쓰기하다.
}
```

#### 멤버 접근 — `의` 조사와 `.` 연산자

구조체 멤버에 접근할 때 한국어의 소유격 조사 `의`를 사용할 수 있습니다.

```글
(* 의 조사로 멤버 접근 — 자연스러운 한국어 *)
철수의 이름을 쓰기하다.
철수의 나이 = 21.

(* 점(.) 연산자 — 간결한 형태 *)
철수.이름을 쓰기하다.
철수.나이 = 21.
```

두 형태는 완전히 동일하게 동작합니다.

### 열거형 (나열)

```글
나열 방향은 {
    위 = 0,
    아래,
    왼쪽,
    오른쪽
}

[시작하기]는 {
    정수 현재방향 = 위.

    현재방향 == 왼쪽이면 {
        "왼쪽으로 이동\n"을 쓰기하다.
    }
}
```

### 합침 (union)

```글
합침 데이터는
    정수 숫자,
    실수 소수,
    문자 글자.
```

### 배열

#### 선언

```글
정수[5] 점수.                          (* 크기 5 배열 *)
정수[3] 값들 = { 10, 20, 30 }.        (* 초기화 *)
문자[256] 버퍼.                        (* 문자 버퍼 *)
```

> **문법**: `타입[크기] 이름` (C의 `타입 이름[크기]`와 다릅니다)

#### 사용

```글
[시작하기]는 {
    정수[5] 점수 = { 90, 85, 78, 92, 88 }.

    합계는 0.
    반복 (정수 번호 = 0: 번호 < 5: 번호++) {
        합계 += 점수[번호].
    }

    "평균: %d\n"을 (합계 / 5)를 쓰기하다.
}
```

## 7. 메모리 관리

### 포인터 선언

```글
정수 값 = 42.
정수 참조 포인터 = &값.     (* 주소 가져오기 *)
```

> `타입 참조` = C의 `타입*`

### 역참조

```글
*포인터 = 100.              (* 포인터가 가리키는 값 변경 *)
```

### 화살표 접근 (→)

```글
학생 참조 포인터 = &철수.
포인터→이름을 쓰기하다.     (* 포인터->이름 *)
포인터→점수 = 100.
```

### 동적 메모리

```글
[시작하기]는 {
    정수 참조 배열 = 할당(100 * 크기(정수)).

    반복 (정수 번호 = 0: 번호 < 100: 번호++) {
        배열[번호] = 번호 * 2.
    }

    배열을 해제하다.
}
```

### 형변환 (로/으로)

조사 `로`/`으로`를 사용하여 형변환을 표현합니다.

```글

정수 코드 = 글자를 정수로 변환하다.
```

## 8. 표준 라이브러리

표준 라이브러리는 **자동으로 포함**됩니다. 별도 `포함` 선언 없이 바로 사용할 수 있습니다.

> 자동 포함을 비활성화하려면 `--no-std` 옵션을 사용하세요.

### 출력 (쓰기)

```글
"안녕하세요\n"을 쓰기하다.                       (* 문자열 출력 *)
"이름: %s, 나이: %d\n"을 이름을 나이를 쓰기하다.  (* 형식 출력 *)
```

형식 지정자:
- `%d` — 정수
- `%lld` — 긴정수
- `%f` — 실수
- `%s` — 문자열
- `%c` — 문자
- `%x` — 16진수
- `%%` — % 기호 자체

### 문자열 함수

```글
정수 길이 = 글자수("안녕하세요").        (* strlen — 문자열 길이 *)
정수 같은가 = 문자열_비교("가", "나").    (* strcmp — 비교 *)
문자열 복사본 = 문자열_복제("원본").      (* strdup — 복제 *)
정수 숫자 = 정수로_변환("42").           (* atoi — 문자열→정수 *)
```

### 메모리 함수

```글
공허 참조 메모리 = 할당(1024).           (* malloc *)
메모리를 해제하다.                        (* free *)
공허 참조 깨끗한 = 초기할당(10, 4).      (* calloc *)
메모리 = 메모리를 2048로 재할당하다.      (* realloc *)
```

### 파일 입출력

```글
공허 참조 파일 = 파일_열기("데이터.txt", "r").

파일이 없으면 {
    "파일 열기 실패\n"을 쓰기하다.
    1로 종료하다.
}

문자[256] 줄.
줄_읽기(줄, 256, 파일).
"%s"를 줄을 쓰기하다.
파일_닫기(파일).
```

### 시스템

```글
"dir"을 명령_실행하다.          (* system() — 시스템 명령 실행 *)
정수 절대 = 절댓값(-42).       (* abs() — 절댓값 *)
0으로 종료하다.                 (* exit() — 프로그램 종료 *)
```

### 표준 라이브러리 모듈

| 모듈 | 설명 | 주요 함수 |
|------|------|----------|
| 기본 | 모든 모듈을 모아놓은 통합 모듈 | (아래 모듈 전체 포함) |
| 입출력 | 표준 입출력 | `쓰기`, `줄_읽기`, `파일_열기`, `파일_닫기` |
| 문자열 | 문자열 조작 | `글자수`, `문자열_비교`, `문자열_복제`, `문자열_연결` |
| 메모리 | 동적 메모리 | `할당`, `해제`, `초기할당`, `재할당` |
| 체계 | 시스템 기능 | `명령_실행`, `종료` |
| 수학 | 수학 함수 | `절댓값` |
| 변환 | 타입 변환 | `정수로_변환` |
| 시간 | 시간 관련 | (시간 측정 등) |

## 10. 고급 기능

### 별칭 (typedef)

```글
별칭 새이름은 기존타입.
별칭 콜백은 [정수, 정수 -> 정수].
```

### 함수 포인터

```글
[정수 왼쪽을 정수 오른쪽을 더하기]는 -> 정수 { 반환 왼쪽 + 오른쪽. }
[정수 왼쪽을 정수 오른쪽을 빼기]는 -> 정수 { 반환 왼쪽 - 오른쪽. }

[시작하기]는 {
    [정수, 정수 -> 정수] 연산 = 더하기.
    "%d\n"을 연산(10, 3)을 쓰기하다.    (* 13 *)

    연산 = 빼기.
    "%d\n"을 연산(10, 3)을 쓰기하다.    (* 7 *)
}
```

함수 포인터 타입 문법: `[인자타입, 인자타입 -> 반환타입]`

### 외부 함수 선언

C 라이브러리 함수를 직접 사용할 때:

```글
외부 [문자열 형식을 ... printf]는 -> 정수.
외부 [문자열 형식을 ... scanf]는 -> 정수.
```

### 가변 인자

```글
[정수 개수를 ... 모두더하기]는 -> 정수 {
    가변목록 목록.
    가변시작(목록, 개수).
    정수 합계 = 0.
    반복 (정수 번호 = 0: 번호 < 개수: 번호++) {
        합계 += 가변인자(목록, 정수).
    }
    가변끝(목록).
    합계를 반환하다.
}

[시작하기]는 {
    정수 결과 = 모두더하기(3, 10, 20, 30).
    "합: %d\n"을 결과를 쓰기하다.    (* 합: 60 *)
}
```

### 조사로 인자 순서 자유롭게

조사가 인자의 역할을 결정하므로, 인자 순서를 바꿔도 동일하게 동작합니다.

```글
[정수 물건값을 정수 할인율로 할인계산]은 -> 정수 {
    물건값 - (물건값 * 할인율 / 100)을 반환하다.
}

[시작하기]는 {
    (* 어떤 순서든 조사가 역할을 결정 *)
    정수 결과1 = 10000을 20으로 할인계산하다.
    정수 결과2 = 20으로 10000을 할인계산하다.
    (* 결과1 == 결과2 == 8000 *)
}
```

## 11. 네이티브 컴파일

### 개요

글 언어는 두 가지 컴파일 경로를 제공합니다:

1. **C 트랜스파일러** (`글도구`): `.글` → C 소스 → C 컴파일러 → 실행파일
2. **네이티브 컴파일러** (`네이티브컴파일러`): `.글` → x86-64 기계어 → PE/ELF 직접 생성

네이티브 컴파일러는 외부 도구(C 컴파일러, 어셈블러, 링커) 없이 `.글` 파일에서 실행파일을 직접 생성합니다.

### 사용법

```
네이티브컴파일러 소스.글                               기본 컴파일 (Windows PE)
네이티브컴파일러 소스.글 -출력 결과.exe                출력 파일명 지정
네이티브컴파일러 주.글 보조1.글 보조2.글 -출력 앱.exe  다중 파일 빌드
네이티브컴파일러 소스.글 --elf                         Linux ELF64 실행파일 생성
네이티브컴파일러 소스.글 --no-std                      표준 라이브러리 자동 포함 생략
네이티브컴파일러 --emit-asm 소스.글                    어셈블리만 생성
네이티브컴파일러 --dump-ir 소스.글                     IR 덤프
```

### 다중 파일 빌드

여러 `.글` 파일을 함께 컴파일할 수 있습니다. 보조 파일들을 먼저 나열하고, 주 파일(`[시작하기]`가 있는 파일)을 마지막에 놓습니다.

```
네이티브컴파일러 수학도구.글 문자열도구.글 주.글 -출력 프로그램.exe
```

### Linux ELF 지원

`--elf` 플래그로 Linux용 ELF64 실행파일을 생성합니다. x86-64 기계어 인코더는 동일하게 사용되며, 바이너리 포맷(PE 대신 ELF)과 시스템 호출 규약만 달라집니다.

```
네이티브컴파일러 소스.글 --elf -출력 프로그램
```

생성된 바이너리는 WSL이나 Linux 환경에서 실행할 수 있습니다.

### 컴파일 파이프라인 비교

```
C 트랜스파일러 (글도구):
  .글 → 파싱 → AST → C 코드 생성 → C 컴파일러(tcc/gcc/cl) → .exe

네이티브 컴파일러:
  .글 → 파싱 → AST → 글IR → x86-64 기계어 → PE/ELF 직접 생성 → .exe
```

## 부록: 새 스타일과 기존 스타일 비교표

다음 표는 같은 동작을 표현하는 두 가지 방법을 보여줍니다. 위쪽이 권장 스타일입니다.

| 동작 | 자연어 스타일 (권장) | 간결/기존 스타일 |
|------|---------------------|----------------|
| 함수 호출 | `10을 20을 더하기하다.` | `10을 20을 더하기다.` 또는 `더하기(10, 20).` |
| 반환 | `결과를 반환하다.` | `반환 결과.` |
| 증가 | `값을 증가하다.` | `값++다.` |
| 감소 | `값을 감소하다.` | `값--다.` |
| 널 검사 | `파일이 없으면 { }` | `파일 == 0이면 { }` |
| 비널 검사 | `파일이 있으면 { }` | `파일 != 0이면 { }` |
| 멤버 접근 | `학생의 이름` | `학생.이름` |
| 문장 연결 | `출력하고 증가하다.` | 두 문장으로 분리 |
| 변수 선언 | `값은 42.` | `정수 값은 42.` |

## 부록: 종합 예제

### 피보나치 수열

```글
[정수 번을 피보나치]는 -> 정수 {
    번이 1보다 작거나같으면 {
        번을 반환하다.
    }
    앞은 번 - 1을 피보나치하다.
    뒤는 번 - 2를 피보나치하다.
    앞 + 뒤를 반환하다.
}

[시작하기]는 {
    번호는 0.
    번호가 10보다 작은동안 {
        "%d "를 피보나치(번호)를 쓰기하고
        번호를 증가하다.
    }
    "\n"을 쓰기하다.
}
```

### 파일에서 읽기

```글
[시작하기]는 {
    공허 참조 파일 = 파일_열기("입력.txt", "r").

    파일이 없으면 {
        "파일을 찾을 수 없습니다.\n"을 쓰기하다.
        1로 종료하다.
    }

    문자[1024] 줄.
    줄_읽기(줄, 1024, 파일)이 있으면 {
        "첫 줄: %s"를 줄을 쓰기하다.
    }

    파일_닫기(파일).
}
```

### 동적 배열 관리

```글
[시작하기]는 {
    용량은 4.
    개수는 0.
    정수 참조 배열 = 할당(크기(정수) * 용량).

    반복 (정수 번호 = 0: 번호 < 8: 번호++) {
        개수가 용량보다 크거나같으면 {
            용량 *= 2.
            배열 = 배열을 (크기(정수) * 용량)으로 재할당하다.
        }
        배열[개수] = (번호 + 1) * 11.
        개수를 증가하다.
    }

    반복 (정수 번호 = 0: 번호 < 개수: 번호++) {
        "%d "를 배열[번호]를 쓰기하다.
    }
    "\n"을 쓰기하다.

    배열을 해제하다.
}
```
