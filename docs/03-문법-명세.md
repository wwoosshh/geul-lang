# 글 언어 - 문법 명세 (Grammar Specification)

## 1. 코드 예제로 보는 문법 체계

### 1.1 Hello World

```글
포함 "표준/입출력"

[시작하기]는
    "안녕, 세계!"를 출력하다.
```

**컴파일 과정 추적**:
```
소스: "안녕, 세계!"를 출력하다.

형태소 분석:
  "안녕, 세계!" → 문자열_리터럴("안녕, 세계!")
  를            → 조사_를 (목적어 역할)
  출력하        → 동사어근("출력") + 동사화접미사("하")
  다            → 어미_다 (서술/종결)
  .             → 마침표 (문장 종결자)

AST:
  호출노드 {
    동사: "출력"
    인자: [("안녕, 세계!", 역할:목적어)]
    어미: 종결
  }

글IR:
  %0 = 전역문자열 "안녕, 세계!"
  호출 @출력(%0)

LLVM IR:
  @.str = private constant [20 x i8] c"\EC\95\88\EB\85\95, \EC\84\B8\EA\B3\84!\00"
  call i32 @글_출력(ptr @.str)
```

### 1.2 변수와 연산

```글
[시작하기]는
    정수 나이는 25다.
    정수 기준은 18이다.

    나이가 기준보다 크면
        "성인"을 출력하다.
    아니면
        "미성년"을 출력하다.
```

**파싱 과정**:
```
"정수 나이는 25다."
  → 선언문 { 타입:정수, 이름:"나이", 주제설정(는), 초기값:25, 종결(다) }

"나이가 기준보다 크면"
  → 조건문 {
      주어: "나이" (가=주어조사),
      비교대상: "기준" (보다=비교조사),
      비교: "크" (형용사어근),
      어미: 면 (조건)
    }
```

### 1.3 함수 정의와 호출 (조사 기반)

```글
(* 함수 정의: 대괄호 안에 매개변수+조사+동사원형 *)
[정수 가를 정수 나에 더하기]는 → 정수
    가에 나를 합하여 반환하다.

[시작하기]는
    정수 결과는 5를 3에 더하기다.
    결과를 출력하다.
```

**핵심: 조사가 인자 역할을 결정**
```
함수 시그니처: [정수 가를 정수 나에 더하기]
  - "가"는 "를" 역할 (목적어 = 피연산 대상)
  - "나"는 "에" 역할 (목적지 = 연산 목표)

호출: 5를 3에 더하기
  - 5 → "를" 역할 → 매개변수 "가"에 바인딩
  - 3 → "에" 역할 → 매개변수 "나"에 바인딩

순서를 바꿔도 동일:
  3에 5를 더하기  ← 조사가 역할을 결정하므로 같은 결과
```

### 1.4 파이프라인 (-어서/-아서 체이닝)

```글
(* 파이프라인: -어서가 결과를 다음 동사의 입력으로 전달 *)
"hello world"를 대문자로바꿔서 앞에서5개잘라서 출력하다.

(* 동일한 코드를 줄바꿈으로 *)
"hello world"를
    대문자로바꿔서
    앞에서5개잘라서
    출력하다.
```

**변환 과정**:
```
파이프라인:
  시작값: "hello world"
  → 대문자로바꾸기 (이전 결과가 암시적 입력)
  → 앞에서5개자르기 (이전 결과가 암시적 입력)
  → 출력하기 (이전 결과가 암시적 입력)

글IR (정규화):
  %0 = 호출 @대문자로바꾸기("hello world")
  %1 = 호출 @앞에서자르기(%0, 5)
  호출 @출력(%1)

LLVM IR:
  %0 = call ptr @글_대문자변환(ptr @.str)
  %1 = call ptr @글_앞에서자르기(ptr %0, i32 5)
  call void @글_출력(ptr %1)
```

### 1.5 반복문

```글
(* while 루프: -동안 어미 *)
정수 합은 0이다.
정수 번호는 1이다.

번호가 100보다 작거나같은동안
    번호를 합에 더하고
    번호에 1을 더하다.

(* for 범위 루프: 부터~까지 *)
번호가 1부터 100까지동안
    번호를 출력하다.
```

### 1.6 구조체 (묶음)

```글
묶음 학생은
    문자열 이름,
    정수 나이,
    실수 평균이다.

[학생 사람의 정보를 출력하기]는
    사람의 이름을 출력하고
    사람의 나이를 출력하다.

[시작하기]는
    학생 철수는 { 이름: "김철수", 나이: 20, 평균: 3.5 }이다.
    철수의 정보를 출력하다.
```

### 1.7 포인터 (참조)

```글
[정수참조 대상의 값을 바꾸기]는
    42를 대상에 넣다.

[시작하기]는
    정수 값은 0이다.
    값의참조의 값을 바꾸다.
    (* 값은 이제 42 *)
    값을 출력하다.
```

### 1.8 배열

```글
[시작하기]는
    정수[5] 점수는 { 90, 85, 78, 92, 88 }이다.

    정수 합은 0이다.
    번호가 0부터 4까지동안
        점수[번호]를 합에 더하다.

    합을 출력하다.
```

### 1.9 주제-평언 스코프 (은/는 컨텍스트)

```글
(* "은/는"으로 주제를 설정하면 이후 생략 가능 *)
파일 보고서는 "결과.txt"를 열어서
    "제목"을 쓰고              (* 보고서에 쓰기 *)
    "내용"을 쓰고              (* 보고서에 쓰기 *)
    닫다.                      (* 보고서를 닫기 *)

(* 위 코드는 아래와 동일 *)
파일 보고서는 "결과.txt"를 열어서
    보고서에 "제목"을 쓰고
    보고서에 "내용"을 쓰고
    보고서를 닫다.
```

### 1.10 병렬 어미 (-며/-면서)

```글
(* -며/-면서는 동시 실행을 의미 *)
자료를 내려받으며 진행률을 표시하다.

(* 이것은 두 작업의 동시 실행으로 변환됨 *)
```

## 2. 어미 활용 완전 체계

### 2.1 종결 어미 (문장을 끝내는 어미)

| 어미 | 의미 | 프로그래밍 동작 | 예시 |
|------|------|----------------|------|
| -다 | 서술/선언 | 문장 종결, 값 반환 | `값을 출력하다.` |
| -라 | 명령 | 즉시 실행 | `출력하라.` |

### 2.2 연결 어미 (문장을 잇는 어미)

| 어미 | 의미 | 프로그래밍 동작 | 예시 |
|------|------|----------------|------|
| -고 | 나열/순차 | 순차 실행 (;) | `더하고 출력하다.` |
| -어서/-아서 | 인과/순차 | 파이프라인 (\|>) | `열어서 읽어서 닫다.` |
| -며/-면서 | 동시 | 병렬 실행 | `받으며 보여주다.` |
| -거나 | 선택 | or / 패턴 매칭 | `찾거나 만들다.` |
| -지만 | 양보 | try-catch / fallback | `열지만 실패하면...` |

### 2.3 전성 어미 (문장 성격을 바꾸는 어미)

| 어미 | 의미 | 프로그래밍 동작 | 예시 |
|------|------|----------------|------|
| -면 | 조건 | if 분기 | `크면 실행하다.` |
| -동안 | 지속 | while 루프 | `참인동안 반복하다.` |
| -도록 | 목적 | assert / ensure | `성공하도록 확인하다.` |
| -ㄹ때 | 시점 | event handler | `클릭할때 실행하다.` |

## 3. 조사 역할 체계

### 3.1 필수 조사

| 조사 | 문법 역할 | 함수에서의 의미 | 사용 맥락 |
|------|----------|----------------|----------|
| 을/를 | 목적어 | 주요 입력 인자 | `값을 처리하다` |
| 이/가 | 주어 | 행위 주체 / 조건 주어 | `값이 크면` |
| 은/는 | 주제 | 변수 선언 / 스코프 설정 | `결과는 0이다` |

### 3.2 부가 조사

| 조사 | 문법 역할 | 함수에서의 의미 | 사용 맥락 |
|------|----------|----------------|----------|
| 에 | 목적지 | 대상/목표 인자 | `배열에 넣다` |
| 로/으로 | 수단 | 변환 방법/타입 | `정수로 바꾸다` |
| 에서 | 출처 | 소스 인자 | `파일에서 읽다` |
| 부터 | 시작점 | 범위 시작 | `1부터` |
| 까지 | 종점 | 범위 끝 | `100까지` |
| 와/과 | 공동 | 추가 인자 / 조인 | `가와 나를 비교하다` |
| 의 | 소유 | 멤버 접근 | `학생의 이름` |
| 보다 | 비교 | 비교 대상 | `기준보다 크면` |

## 4. 타입 체계

### 4.1 기본 타입

| 글 타입 | C 대응 | LLVM IR | 크기 |
|---------|--------|---------|------|
| 정수 | int | i32 | 4바이트 |
| 긴정수 | long long | i64 | 8바이트 |
| 짧은정수 | short | i16 | 2바이트 |
| 작은정수 | char (숫자) | i8 | 1바이트 |
| 부호없는정수 | unsigned int | i32 | 4바이트 |
| 실수 | double | double | 8바이트 |
| 짧은실수 | float | float | 4바이트 |
| 문자 | char | i8 | 1바이트 |
| 문자열 | char* | ptr | 8바이트 |
| 참거짓 | _Bool | i1 | 1바이트 |
| 공허 | void | void | 0 |

### 4.2 복합 타입

```
타입참조     = 타입 + "참조"        예: 정수참조 (= int*)
배열타입     = 타입 + [크기]        예: 정수[10] (= int[10])
묶음타입     = 묶음 이름 { ... }    예: 묶음 점 { 정수 가로, 정수 세로 }
나열타입     = 나열 이름 { ... }    예: 나열 색 { 빨강, 파랑, 초록 }
함수참조     = [인자타입들 → 반환타입]  예: [정수, 정수 → 정수]
```
