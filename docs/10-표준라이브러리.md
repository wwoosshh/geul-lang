# 글 표준 라이브러리 명세

## 설계 원칙

1. **순수 한글 인터페이스** — 사용자 코드에서 영어 함수명 노출 금지
2. **C 함수 래핑** — 내부적으로 C 표준 함수를 호출하되, 한글 래퍼로 캡슐화
3. **최소 의존성** — 각 모듈은 독립적으로 포함 가능
4. **외부 함수 격리** — `외부` 선언은 표준 라이브러리 내부에서만 존재, 사용자에게 비노출

## 모듈 목록

### 표준/입출력.글

콘솔 및 파일 입출력.

```
포함 "표준/입출력.글"
```

| 함수 | 시그니처 | 설명 | C 함수 |
|------|---------|------|--------|
| 쓰기 | 가변인자 (printf 래퍼) | 서식 문자열 출력 | printf |
| 쓰기_줄 | `[문자열 글을 쓰기_줄]는 → 정수` | 문자열 + 줄바꿈 출력 | puts |
| 파일_열기 | `[문자열 경로를 문자열 모드로 파일_열기]는 → 공허 참조` | 파일 열기 | fopen |
| 파일_닫기 | `[공허 참조 파일을 파일_닫기]는 → 정수` | 파일 닫기 | fclose |
| 파일_읽기 | `[공허 참조 버퍼에 정수 크기를 정수 개수를 공허 참조 파일에서 파일_읽기]는 → 정수` | 파일 읽기 | fread |
| 파일_쓰기 | `[공허 참조 버퍼에서 정수 크기를 정수 개수를 공허 참조 파일에 파일_쓰기]는 → 정수` | 파일 쓰기 | fwrite |
| 줄_읽기 | `[문자열 버퍼에 정수 크기를 공허 참조 파일에서 줄_읽기]는 → 문자열` | 한 줄 읽기 | fgets |
| 파일_위치이동 | `[공허 참조 파일을 긴정수 위치로 정수 기준에서 파일_위치이동]는 → 정수` | 파일 위치 이동 | fseek |
| 파일_위치 | `[공허 참조 파일을 파일_위치]는 → 긴정수` | 현재 파일 위치 | ftell |

| 파일_글넣기 | `[문자열 글을 공허 참조 파일에 파일_글넣기]는 → 정수` | 문자열 출력 (파일) | fputs |
| 파일에_쓰기 | 가변인자 (fprintf 래퍼) | 파일에 서식 출력 | fprintf |
| 버퍼에_쓰기 | 가변인자 (snprintf 래퍼) | 버퍼에 서식 출력 | snprintf |
| 출력_전환 | `정의 출력_전환 freopen.` | 출력 리다이렉트 | freopen |
| 파일_삭제 | `정의 파일_삭제 remove.` | 파일 삭제 | remove |
| 표준입력/표준출력/표준오류 | `정의` 매크로 | 표준 스트림 상수 | stdin/stdout/stderr |

**사용 예시:**
```

```

## 라이브러리 작성 방법

새 표준 라이브러리 모듈을 만들려면:

1. `표준/` 디렉토리에 `.글` 파일 생성
2. C 함수를 `외부` 선언
3. 한글 래퍼 함수 작성

```
(* 표준/수학.글 예시 *)
외부 [실수 값을 sqrt]는 → 실수.
외부 [실수 값을 sin]는 → 실수.
외부 [실수 값을 cos]는 → 실수.

[실수 값을 제곱근]는 → 실수 {
    반환 값을 sqrt다.
}

[실수 값을 사인]는 → 실수 {
    반환 값을 sin다.
}

[실수 값을 코사인]는 → 실수 {
    반환 값을 cos다.
}
```
