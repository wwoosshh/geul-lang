# 글 프로그래밍 언어 — 문법 레퍼런스 v0.5

> 이 문서는 네이티브 컴파일러 v0.5에서 **실제 동작이 검증된 문법**만을 기술합니다.

---

## 1. 기본 구조

### 프로그램 진입점

```글
포함 "std.gl"

[시작하기]는 {
    "안녕 세계\n"을 쓰다.
}
```

- `포함 "std.gl"` — 표준 라이브러리 포함 (필수)
- `[시작하기]는 { }` — 프로그램 시작점 (C의 `main()`)
- 모든 문장은 **마침표(`.`)**로 끝남

### 주석

```글
(* 이것은 주석입니다 *)
(* 중첩 (* 주석도 *) 가능합니다 *)
```

### 실행 방법

```
네이티브컴파일러.exe 소스파일.글
```
현재 디렉토리에 `소스파일.exe`가 생성됩니다. 출력 경로를 지정하려면:
```
네이티브컴파일러.exe 소스파일.글 -o 출력.exe
```

## 2. 타입과 변수

### 기본 타입

| 타입 | 설명 | C 대응 |
|------|------|--------|
| `정수` | 64비트 정수 | `long long` |
| `긴정수` | 64비트 정수 | `long long` |
| `짧은정수` | 16비트 정수 | `short` |
| `작은정수` | 8비트 정수 | `char` |
| `실수` | 64비트 부동소수점 | `double` |
| `짧은실수` | 32비트 부동소수점 | `float` |
| `문자` | 8비트 문자 | `char` |
| `문자열` | 문자 포인터 | `char*` |
| `참거짓` | 정수 (0/1) | `int` |
| `공허` | 반환값 없음 | `void` |

### 변수 선언

```글
정수 나이 = 25.
문자열 이름 = "철수".
실수 온도 = 36.5.
참거짓 완료 = 0.
정수 값.                (* 초기값 없이 선언 *)
```

### 전역 변수

함수 밖에서 선언하면 전역 변수가 됩니다.

```글
포함 "std.gl"
정수 카운터 = 0.

[시작하기]는 {
    카운터 = 42.
    "%d\n"을 카운터를 쓰다.
}
```

### 상수

```글
상수 정수 최대값 = 100.
```

## 3. 함수

### 함수 정의

```글
(* 매개변수 없는 함수 *)
[인사]는 {
    "안녕하세요\n"을 쓰다.
}

(* 반환값이 있는 함수 *)
[더블]은 → 정수 {
    반환 42.
}

(* 매개변수가 있는 함수 — 조사가 역할을 결정 *)
[정수 수를 제곱]은 → 정수 {
    반환 수 * 수.
}

(* 다중 매개변수 — 조사로 구분 *)
[정수 왼쪽을 정수 오른쪽에 더하기]는 → 정수 {
    반환 왼쪽 + 오른쪽.
}
```

### 함수 호출

**SOV(주어-목적어-동사) 호출:**
```글
5를 제곱하다.                    (* 결과: 25 *)
5를 3에 더하다.                  (* 결과: 8 *)
"안녕\n"을 쓰다.                 (* printf 호출 *)
```

**직접 호출 (C 스타일):**
```글
제곱(5).
더하기(5, 3).
printf("안녕\n").
```

### 반환

```글
반환 42.                        (* 직접 반환 *)
값을 반환하다.                   (* SOV 스타일 반환 *)
```

### main 매개변수

```글
[정수 argc를 문자열 참조 argv를 시작하기]는 → 정수 {
    "인자 수: %d\n"을 argc를 쓰다.
    반환 0.
}
```

## 4. 제어 흐름

### 조건문 (이면/아니면)

```글
값 > 0이면 {
    "양수\n"을 쓰다.
}
아니면 값 == 0이면 {
    "영\n"을 쓰다.
}
아니면 {
    "음수\n"을 쓰다.
}
```

### while 반복 (동안)

```글
정수 번호 = 0.
(번호 < 10)동안 {
    "%d\n"을 번호를 쓰다.
    번호를 증가하다.
}
```

### for 반복 (반복)

구분자로 **쌍점(`:`)** 사용 (세미콜론 아님):

```글
반복 (정수 번호 = 0: 번호 < 10: 번호++) {
    "%d\n"을 번호를 쓰다.
}
```

### do-while (반복하기~동안)

```글
정수 횟수 = 0.
반복하기 {
    횟수++.
} (횟수 < 3)동안.
```

### switch (갈래/경우/기본)

```글
갈래 (요일) {
    경우 1: { "월요일\n"을 쓰다. }
    경우 2: { "화요일\n"을 쓰다. }
    경우 3: { "수요일\n"을 쓰다. }
    기본: { "기타\n"을 쓰다. }
}
```

### 탈출/계속

```글
(조건)동안 {
    문제이면 { 탈출. }     (* break *)
    건너뛸것이면 { 계속. }  (* continue *)
}
```

## 5. 연산자

### 산술 연산자

```글
정수 합 = 10 + 3.
정수 차 = 10 - 3.
정수 곱 = 10 * 3.
정수 몫 = 10 / 3.
정수 나머지 = 10 % 3.
```

### 비교 연산자

```글
값 == 10이면 { }        (* 같음 *)
값 != 10이면 { }        (* 다름 *)
값 > 10이면 { }         (* 큼 *)
값 < 10이면 { }         (* 작음 *)
값 >= 10이면 { }        (* 크거나 같음 *)
값 <= 10이면 { }        (* 작거나 같음 *)
```

### 논리 연산자

글 언어에서 논리 AND는 `*`, OR는 `+`로 표현합니다:

```글
(값 > 0) * (값 < 100)이면 { }   (* AND: 둘 다 참 *)
(값 == 0) + (값 == 1)이면 { }   (* OR: 하나라도 참 *)
```

### 복합 대입 연산자

```글
합계 += 10.
합계 -= 5.
합계 *= 2.
합계 /= 3.
합계 %= 7.
```

### 증감 연산자

```글
번호++.                  (* 후위 증가 *)
번호--.                  (* 후위 감소 *)
번호를 증가하다.          (* 동사형 증가 *)
번호를 감소하다.          (* 동사형 감소 *)
```

### 삼항 연산자

```글
정수 최대 = 값1 > 값2 ? 값1 : 값2.
```

### 비트 연산자

```글
정수 결과 = 값 & 255.       (* AND *)
정수 결과 = 값 >> 8.        (* 오른쪽 시프트 *)
정수 결과 = 값 << 4.        (* 왼쪽 시프트 *)
```

## 6. 배열

### 선언

타입 뒤에 크기를 지정합니다 (C와 순서가 다름):

```글
정수[10] 점수.              (* C: int 점수[10]; *)
문자열[100] 이름목록.       (* C: char* 이름목록[100]; *)
문자[256] 버퍼.             (* C: char 버퍼[256]; *)
```

### 접근과 대입

```글
점수[0] = 90.
점수[1] = 85.
정수 첫점수 = 점수[0].
```

### 반복문과 함께 사용

```글
정수[5] 값.
값[0] = 50. 값[1] = 30. 값[2] = 40. 값[3] = 10. 값[4] = 20.

반복 (정수 번호 = 0: 번호 < 5: 번호++) {
    "%d "를 값[번호]를 쓰다.
}
```

## 7. 구조체, 열거형, 공용체

### 구조체 (묶음)

정의는 쉼표와 마침표를 사용합니다 (중괄호 아님):

```글
묶음 좌표는 정수 엑스, 정수 와이.
```

사용:
```글
좌표 위치.
위치의 엑스 = 10.
위치의 와이 = 20.
"%d, %d\n"을 위치의 엑스를 위치의 와이를 쓰다.
```

### 열거형 (나열)

```글
나열 색깔은 { 빨강, 초록, 파랑 }

나열 방향은 { 북 = 0, 동, 남, 서 }
```

사용:
```글
정수 내색 = 초록.   (* 값: 1 *)
```

### 공용체 (합침)

```글
합침 값은 정수 정수값, 실수 실수값.
```

## 10. 고급 기능

### 별칭 (typedef)

```글
별칭 크기 = 정수.
별칭 바이트 = 문자.
```

### 외부 선언 (extern)

```글
외부 [문자열 이름을 정수 값을 외부함수]는 → 정수.
```

### 정적 변수/함수 (static)

```글
정적 정수 내부카운터 = 0.
정적 [도우미]는 { }
```

### 함수 포인터

```글
[정수, 정수 → 정수] 연산.     (* int (*연산)(int, int) *)
연산 = 더하기.
정수 결과 = 연산(3, 5).
```

### 가변 인자

```글
가변목록 인자.
가변시작(인자, 마지막매개변수).
정수 값 = 가변인자(인자, 정수).
가변끝(인자).
```

### 형변환

```글
문자열 텍스트 = 값으로 문자열하다.   (* (char*)값 *)
정수 주소 = 포인터를 정수로하다.     (* (long long)포인터 *)
```

## 13. 주의사항

### 배열 선언 순서

C와 다릅니다:
```글
정수[50] 이름.              (* 글: 타입[크기] 이름 *)
```
```c
int 이름[50];               /* C:  타입 이름[크기] */
```

### for 루프 구분자

**쌍점(`:`)** 을 사용합니다:
```글
반복 (정수 번호 = 0: 번호 < 10: 번호++) { }
```

### 논리 연산자

`&&`와 `||` 대신 `*`(AND)와 `+`(OR)를 사용합니다:
```글
(조건1) * (조건2)이면 { }      (* 조건1 AND 조건2 *)
(조건1) + (조건2)이면 { }      (* 조건1 OR 조건2 *)
```

### 구조체 정의

중괄호가 아닌 **쉼표와 마침표**를 사용합니다:
```글
묶음 이름은 타입1 필드1, 타입2 필드2.
```

### 변수명 주의

`자`로 끝나는 변수명 (`대문자`, `소문자` 등)은 `자`가 어미로 분리될 수 있어 예기치 않은 동작을 할 수 있습니다. 다른 이름을 사용하세요.
