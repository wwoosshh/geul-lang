# 글 언어 VS Code 확장 설치 방법

## 1단계: VS Code 설치
https://code.visualstudio.com 에서 무료 다운로드

## 2단계: 글 확장 설치

### 방법 A: 폴더 복사 (가장 간단)
1. 이 폴더(`글-vscode`)를 다음 위치에 복사:
   - Windows: `%USERPROFILE%\.vscode\extensions\geul-language-0.1.0`
   - macOS: `~/.vscode/extensions/geul-language-0.1.0`
   - Linux: `~/.vscode/extensions/geul-language-0.1.0`
2. VS Code 재시작

### 방법 B: VSIX 패키지 (배포용)
```
npm install -g @vscode/vsce
cd 글-vscode
npm init -y
vsce package
```
생성된 `.vsix` 파일을 VS Code에서 설치:
`확장 → ... → VSIX에서 설치`

## 3단계: 글도구 설정
1. 글도구.exe를 원하는 위치에 복사
2. VS Code 설정 → "글 언어" → 글도구 경로 지정
   또는 글도구.exe를 시스템 PATH에 추가

## 사용법

| 단축키 | 기능 |
|--------|------|
| **F5** | 현재 .글 파일 컴파일 + 실행 |
| **Ctrl+Shift+B** | 빌드 (.exe 생성) |
| **Ctrl+Shift+C** | 문법 검사 |

### 코드 자동완성 (스니펫)
에디터에서 다음을 입력하고 Tab:
- `시작` → 시작하기 함수 템플릿
- `함수` → 함수 정의 템플릿
- `이면` → 조건문
- `동안` → 반복문
- `쓰기` → printf 출력
- `포함` → 표준 라이브러리 포함

### 기능
- **문법 강조**: .글 파일을 열면 자동으로 색상 표시
- **자동 괄호**: `{`, `(`, `[`, `"` 자동 닫기
- **오류 표시**: 저장 시 문법 오류를 빨간 밑줄로 표시
- **터미널 실행**: F5 누르면 내장 터미널에서 실행
