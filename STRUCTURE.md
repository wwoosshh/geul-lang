## 빌드 파이프라인

```
사용자 .글 소스
    ↓
네이티브컴파일러.exe  ← 사용자가 사용하는 컴파일러
    ↓
.exe (Windows PE)

네이티브컴파일러.exe 빌드 과정:
    자체호스팅/*.글 → 글cc(부트스트랩) → .c → MSVC → 네이티브컴파일러.exe
```

## 디렉토리 규칙

| 디렉토리 | 내용 | Git 추적 |
|----------|------|----------|
| 자체호스팅/ | 컴파일러 소스 (.글) | ✓ |
| 부트스트랩/ | C 부트스트랩 소스 | ✓ (src/, include/) |
| 표준/ | 표준 라이브러리 (.글) | ✓ |
| 예제/ | 예제 소스 (.글만) | ✓ |
| 배포/ | 빌드 결과물 (.exe) | ✗ (릴리즈로 배포) |
| 벤치마크/ | 벤치마크 소스+결과 | ✓ |
| docs/ | 문서 | ✓ |
| 도구/ | 개발 도구 소스 | ✓ (바이너리 제외) |

**빌드 잔여물(.obj, .exe, .c)은 Git에 포함하지 않습니다.**
